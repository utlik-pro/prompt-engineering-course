Cmd+K — быстрые команды продуктивности

Мгновенное редактирование текста, планов и данных без открытия чата

Заметка для преподавателя

Цель урока: Научить студентов использовать Cmd+K как самый быстрый инструмент для точечного редактирования текстов, планов и данных прямо на месте.

Ключевой момент: Cmd+K идеально подходит для локальных правок. Студенты должны понять разницу между Cmd+K, Chat и Composer и выбирать правильный инструмент под задачу.

Практика: Подготовьте несколько текстов разного типа (письмо, список задач, таблица) и проработайте Cmd+K на каждом из них.

Что такое Cmd+K?

Cmd+K (или Ctrl+K на Windows) — это самый быстрый способ изменить текст прямо на месте. Выделяете фрагмент, нажимаете Cmd+K, пишете команду — и текст мгновенно трансформируется.

Выделяете текст → Cmd+K → Даёте команду → Готово!

В отличие от Chat (диалог) и Composer (создание с нуля), Cmd+K работает точечно: вы указываете конкретный фрагмент и говорите, что с ним сделать. Никаких лишних шагов.

Как использовать Cmd+K
1
Выделите текст

Мышкой или Shift+стрелки. Можно выделить слово, абзац, список или весь документ

2
Нажмите Cmd+K (Ctrl+K на Windows)

Появится компактное поле ввода прямо над выделенным текстом

3
Напишите команду на естественном языке

Например: "переписать формальнее", "сократить до 3 предложений", "перевести на английский"

4
Нажмите Enter и примите изменения

ИИ покажет diff (разницу). Нажмите Tab или кнопку Accept, чтобы применить

Без выделения: Если не выделить текст, Cmd+K создаст новый текст в позиции курсора. Это удобно для быстрой вставки.
Cmd+K для текстов

Cmd+K превращает вас в мастера редактирования. Вот примеры команд для работы с текстами:

"переписать формальным деловым тоном"

Превращает неформальный текст в деловое письмо. Идеально для рабочей переписки.

"сократить до 3 предложений, сохранив суть"

Убирает воду, оставляет ключевые мысли. Работает для длинных абзацев и описаний.

"перевести на английский"

Мгновенный перевод выделенного фрагмента. Работает с любыми языками.

"исправить грамматику и пунктуацию"

Автоматическая корректура. Исправляет ошибки, не меняя смысл.

"сделать более убедительным"

Усиливает аргументацию, добавляет уверенности. Полезно для презентаций и предложений.

"упростить для широкой аудитории"

Заменяет сложные термины простыми словами. Делает текст доступным.

Пример трансформации:
ДО (неформально):

Привет! Слушай, мне нужно больше времени на проект, не успеваю. Давай сдвинем дедлайн?

ПОСЛЕ (формально):

Добрый день. В связи с увеличением объёма работ прошу рассмотреть возможность переноса срока сдачи проекта.

Cmd+K для планов и списков

Планы и списки задач — идеальная территория для Cmd+K. Вот что можно делать:

"отсортировать по приоритету: критичные сверху"

ИИ проанализирует задачи и перестроит список по важности.

"добавить реалистичные дедлайны к каждому пункту"

ИИ оценит сложность каждой задачи и предложит сроки.

"конвертировать в Markdown-таблицу с колонками: задача, приоритет, дедлайн, статус"

Превращает обычный список в структурированную таблицу.

"разбить каждый пункт на подзадачи"

Декомпозирует крупные задачи на конкретные шаги.

"добавить чекбоксы Markdown к каждому пункту"

Превращает список в интерактивный чек-лист: - [ ] задача

Пример: список в таблицу
До: простой список 
- Подготовить презентацию
- Позвонить клиенту
- Написать отчёт
- Обновить сайт
- Провести встречу с командой
После: Cmd+K "конвертировать в таблицу с приоритетами и дедлайнами" 
| Задача                      | Приоритет | Дедлайн    | Статус     |
|-----------------------------|-----------|------------|------------|
| Позвонить клиенту           | Высокий   | Сегодня    | - [ ]      |
| Провести встречу с командой | Высокий   | Завтра     | - [ ]      |
| Подготовить презентацию     | Средний   | Пятница    | - [ ]      |
| Написать отчёт              | Средний   | Пятница    | - [ ]      |
| Обновить сайт               | Низкий    | След. нед. | - [ ]      |
Cmd+K для данных

Cmd+K отлично справляется с быстрой обработкой данных прямо в редакторе:

"преобразовать JSON в читаемую Markdown-таблицу"

Превращает сырые данные в удобный для чтения формат.

"извлечь ключевые тезисы из этого текста"

Из длинного текста создаёт краткий список главных идей.

"создать executive summary на основе этих данных"

Из набора фактов и цифр формирует связное резюме для руководства.

"преобразовать в CSV-формат"

Конвертирует таблицу или список в формат для Excel/Google Sheets.

"добавить итоги и средние значения"

Для числовых данных рассчитывает суммы и статистику.

Cmd+K vs Chat vs Composer: когда что использовать

Три мощных инструмента в Cursor — каждый для своей задачи. Вот как выбирать:

Характеристика Cmd+K Chat Composer
Лучше всего для Точечное редактирование выделенного фрагмента Вопросы, обсуждения, анализ Создание и редактирование нескольких файлов
Скорость Мгновенно Средняя Медленнее, но мощнее
Контекст Только выделенный текст Текущий файл + подключённые Весь проект
Результат Изменяет текст на месте Ответ в панели чата Изменения в файлах проекта
Пример задачи Переписать абзац формальнее Как лучше структурировать план? Создать систему из 5 файлов
Горячая клавиша Cmd+K / Ctrl+K Cmd+L / Ctrl+L Cmd+I / Ctrl+I
Правило выбора: Нужно быстро изменить конкретный кусок текста? Cmd+K. Нужно обсудить или спросить? Chat. Нужно создать что-то масштабное? Composer.
Горячие клавиши мастера продуктивности

Запомните эти сочетания — они сэкономят вам часы работы:

Cmd+K / Ctrl+K

Быстрое редактирование (Inline Edit)

Cmd+L / Ctrl+L

Открыть AI Chat

Cmd+I / Ctrl+I

Открыть Composer

Cmd+Shift+K

Удалить строку целиком

Cmd+D

Выделить следующее совпадение слова

Cmd+Shift+L

Выделить все совпадения слова

Alt+Up / Alt+Down

Переместить строку вверх/вниз

Cmd+Shift+P

Палитра команд (все действия)

Cmd+P

Быстрое открытие файла по имени

Cmd+B

Скрыть/показать боковую панель

Cmd+Z / Cmd+Shift+Z

Отменить / Повторить действие

Cmd+S

Сохранить файл

На Windows: Везде, где написано Cmd, используйте Ctrl. Например: Ctrl+K, Ctrl+L, Ctrl+I.
Практика: 10 упражнений на Cmd+K

Выполните все 10 упражнений. Для каждого создайте текст в файле, выделите его и примените Cmd+K.

Упражнение 1: Формальное письмо

Напишите неформальное сообщение коллеге. Выделите его → Cmd+K → "переписать как деловое письмо"

Упражнение 2: Сокращение

Напишите длинный абзац (5-6 предложений). Выделите → Cmd+K → "сократить до 2 предложений"

Упражнение 3: Перевод

Напишите абзац на русском. Выделите → Cmd+K → "перевести на английский"

Упражнение 4: Список в таблицу

Создайте маркированный список из 5 задач. Выделите → Cmd+K → "конвертировать в Markdown-таблицу"

Упражнение 5: Приоритизация

Список из 7 задач в случайном порядке. Выделите → Cmd+K → "отсортировать по приоритету"

Упражнение 6: Дедлайны

Список задач без дат. Выделите → Cmd+K → "добавить реалистичные дедлайны начиная с сегодня"

Упражнение 7: Грамматика

Напишите текст с намеренными ошибками. Выделите → Cmd+K → "исправить грамматику и пунктуацию"

Упражнение 8: Извлечение тезисов

Вставьте длинный текст (можно из интернета). Выделите → Cmd+K → "извлечь 5 ключевых тезисов"

Упражнение 9: Смена тона

Напишите серьёзный текст. Выделите → Cmd+K → "переписать с юмором". Затем → "переписать вдохновляюще"

Упражнение 10: Конвертация формата

Создайте простую таблицу в Markdown. Выделите → Cmd+K → "конвертировать в CSV-формат"

Совет: После каждого Cmd+K вы можете отменить изменения через Cmd+Z. Экспериментируйте без страха!
