Установка и настройка Cursor

Устанавливаем Cursor, создаём рабочее пространство "Моя жизнь" и настраиваем ИИ под себя

Заметка для преподавателя

Цель урока: Каждый студент должен установить Cursor, создать рабочее пространство с папками и написать свой первый .cursorrules файл.

Важно: Показывайте экран и делайте медленно. Помогайте тем, у кого проблемы с установкой. Убедитесь, что все выбрали ИИ-модель и могут открыть чат.

Шаг 1: Установка Cursor

Cursor — это редактор на базе VS Code со встроенным ИИ-ассистентом. Установка занимает 5 минут на любой системе.

Официальный сайт: cursor.sh

Установка на Windows:

1
Перейдите на cursor.sh и нажмите "Download"

Сайт автоматически определит Windows и предложит .exe файл

2
Запустите установщик CursorSetup.exe

Следуйте стандартному процессу: Next → Next → Install

3
Запустите Cursor из меню Пуск

Cursor появится в списке программ после установки

Установка на Mac:

1
Скачайте .dmg файл с cursor.sh

Сайт предложит версию для вашего процессора (Intel или Apple Silicon)

2
Откройте .dmg и перетащите Cursor в Applications

Стандартная установка для Mac — drag & drop в папку Applications

3
Запустите из Launchpad или Spotlight (Cmd+Space → "Cursor")

При первом запуске Mac может попросить разрешение — нажмите "Открыть"

Установка на Linux:

1
Скачайте .AppImage файл с cursor.sh

Для Debian/Ubuntu также доступен .deb пакет

2
Сделайте файл исполняемым и запустите

chmod +x Cursor.AppImage && ./Cursor.AppImage

Регистрация: При первом запуске создайте аккаунт (можно через Google). Бесплатный план включает 2000 запросов к ИИ в месяц — этого достаточно для обучения.
Шаг 2: Обзор интерфейса

Разберём основные области Cursor. Интерфейс на английском, но мы объясним каждый элемент:

📁

Explorer (левая панель)

Ваши файлы и папки
Здесь отображается структура проекта. Кликайте на файлы, чтобы открыть их в редакторе. Правый клик — создание новых файлов и папок.

📝

Editor (центр)

Основная рабочая область
Здесь вы редактируете файлы — текст, заметки, планы. Можно открыть несколько файлов во вкладках и работать с ними параллельно.

🤖

AI Chat (правая панель)

Ваш ИИ-ассистент
Открывается по Cmd+L. Сюда вы пишете запросы к ИИ. Он видит ваши открытые файлы и может работать с контекстом проекта.

⌨️

Terminal (нижняя панель)

Командная строка
Открывается по Ctrl+`. Полезна для запуска скриптов и автоматизации. Мы будем использовать её в продвинутых уроках.

Шаг 3: Настройка ИИ-модели

Cursor позволяет выбирать между разными ИИ-моделями. Каждая имеет свои сильные стороны:

Модель Сильные стороны Рекомендация
Claude 3.5 Sonnet Глубокий анализ, длинные тексты, рассуждения Отлично для планирования и анализа
GPT-4o Универсальность, креативность, скорость Хорош для брейнсторминга и контента
Claude 3.5 Haiku Быстрые ответы, простые задачи Для быстрых вопросов и редактирования
GPT-4o Mini Скорость, экономия запросов Для простых задач, если запросы ограничены

Как выбрать модель:

1
Откройте чат (Cmd+L / Ctrl+L)

В нижней части окна чата вы увидите название текущей модели

2
Кликните на название модели

Откроется выпадающий список с доступными моделями

3
Выберите Claude 3.5 Sonnet (рекомендуем для начала)

Это лучшая модель для аналитических и текстовых задач

Совет: Вы можете менять модель для каждого запроса. Используйте Claude для анализа и планирования, GPT-4o для креативных задач.
Шаг 4: Создаём рабочее пространство "Моя жизнь"

Сейчас мы создадим структуру папок, которая станет вашим цифровым центром управления жизнью. Это ваш "второй мозг" в Cursor.

Шаг 1: Создаём корневую папку

  • Создайте на рабочем столе (или в удобном месте) папку: My-Life-System
  • В Cursor: File → Open Folder → выберите эту папку

Шаг 2: Создаём структуру папок

Создайте следующие папки через левую панель (правый клик → New Folder):

Структура папок 
My-Life-System/
├── projects/          # Активные проекты и задачи
├── notes/             # Заметки, идеи, конспекты
├── plans/             # Планы: дневные, недельные, месячные, годовые
├── templates/         # Шаблоны для повторяющихся задач
├── data/              # Данные: бюджет, трекеры, списки
├── journal/           # Дневник и рефлексия
└── .cursorrules       # Системный промпт для ИИ (файл!)

Шаг 3: Создаём первые файлы

В каждой папке создайте стартовый файл:

📋
projects/active-projects.md

Список текущих проектов с описанием и статусами

📓
plans/weekly-plan.md

Шаблон недельного плана

💡
notes/ideas.md

Место для быстрых идей и мыслей

💰
data/budget.md

Бюджет и учёт расходов

Шаг 5: Настройка .cursorrules — ваш системный промпт

Файл .cursorrules — это самая мощная функция Cursor для промпт-инженера. Это системный промпт, который ИИ читает при каждом обращении. Вы настраиваете его один раз — и ИИ всегда знает ваш контекст.

Создайте файл .cursorrules в корне проекта:

.cursorrules 
# Системные инструкции для ИИ-ассистента

## Кто я
Ты — мой персональный ИИ-ассистент для управления жизнью и проектами.
Ты помогаешь мне планировать, анализировать, принимать решения и
организовывать информацию.

## Как отвечать
- Отвечай на русском языке
- Будь конкретным и практичным
- Предлагай структурированные решения (списки, таблицы)
- Если задача сложная — разбивай на шаги
- Используй Markdown-форматирование для читаемости
- Всегда думай пошагово (Chain of Thought)

## Контекст моей жизни
- Папка /projects — мои активные проекты
- Папка /plans — мои планы и цели
- Папка /notes — заметки и идеи
- Папка /data — данные, бюджет, трекеры
- Папка /journal — дневник
- Папка /templates — шаблоны

## Правила работы
- При планировании учитывай мои текущие проекты из /projects
- При анализе ссылайся на конкретные файлы
- Предлагай создавать новые файлы, если информацию стоит сохранить
- Не удаляй информацию без моего подтверждения
Это ваш системный промпт! Вспомните урок о System Prompts — здесь вы применяете те же принципы. ИИ будет следовать этим инструкциям в каждом диалоге, не нужно повторять контекст.
Важно: Файл .cursorrules должен быть именно в корне папки проекта (в My-Life-System, не внутри подпапок). Cursor автоматически его подхватит.
Шаг 6: Горячие клавиши — ваши суперспособности

Запомните эти 6 комбинаций — они покроют 90% ваших действий в Cursor:

Действие Mac Windows / Linux Зачем
Открыть чат Cmd + L Ctrl + L Основной способ общения с ИИ
Открыть Composer Cmd + I Ctrl + I Мультифайловое редактирование с ИИ
Inline Edit (Cmd+K) Cmd + K Ctrl + K Быстрое редактирование выделенного текста
Палитра команд Cmd + Shift + P Ctrl + Shift + P Доступ ко всем командам Cursor
Сохранить файл Cmd + S Ctrl + S Сохранить текущий файл
Быстрый поиск файла Cmd + P Ctrl + P Быстро найти и открыть любой файл в проекте
Мнемоника: L = Chat (от "Language"), I = Composer (от "Instrument"), K = быстрое редактирование (от "Keyboard edit"). Потренируйтесь прямо сейчас — откройте и закройте каждую панель 3 раза.
Дополнительные настройки для комфорта
Настройка Где найти Рекомендация
Тема оформления Cmd+Shift+P → "Color Theme" Выберите тёмную или светлую по вкусу
Размер шрифта Settings → Font Size 14-16 px для комфортного чтения
Автосохранение File → Auto Save Обязательно включите! Никогда не потеряете работу
Перенос строк View → Word Wrap Включите — текст не будет уходить за экран
Язык интерфейса Cmd+Shift+P → "Display Language" Можно установить русский через расширение
