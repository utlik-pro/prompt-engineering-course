Что можно автоматизировать через Cursor

Если вы делаете что-то больше двух раз — это можно автоматизировать. Cursor позволяет создавать простые скрипты и шаблоны даже без опыта программирования. Вы описываете задачу словами, а ИИ создаёт решение.

📝
Обработка текстов

Форматирование, перевод, рерайт, извлечение данных, генерация из шаблонов

📊
Генерация отчётов

Еженедельные обзоры, аналитика, подведение итогов, сводки

✉️
Шаблоны писем

Холодные письма, ответы клиентам, рассылки, уведомления

🔄
Конвертация форматов

CSV в Markdown-таблицы, списки в чеклисты, данные в отчёты

Ключевая мысль: вам не нужно быть программистом. Вы просите Cursor создать скрипт, а он делает всё за вас. Вам остаётся лишь запустить его.

Скрипты без программирования

Cursor может написать для вас простой скрипт на Python, который выполнит рутинную задачу. Вам не нужно понимать код — достаточно описать, что нужно сделать.

Пример 1: Переименование файлов

У вас 50 фотографий с названиями IMG_2847.jpg, IMG_2848.jpg... Хотите переименовать в "Грузия-день1-001.jpg"

Промпт в Chat 
Напиши Python-скрипт, который:
1. Берёт все .jpg файлы из папки photos/
2. Сортирует их по дате создания
3. Переименовывает по шаблону:
   "Грузия-день{N}-{номер}.jpg"
   где N — день (каждые 24 часа = новый день),
   номер — порядковый номер фото за день (001, 002...)
4. Выводит на экран: было → стало

Сохрани скрипт как rename-photos.py
Я не программист — добавь комментарии
и инструкцию как запустить.
Результат: rename-photos.py (упрощённо) 
# Скрипт для переименования фотографий
# Как запустить: откройте терминал в Cursor и введите:
#   python rename-photos.py

import os
from datetime import datetime

# Папка с фотографиями
PHOTO_DIR = "photos/"
# Шаблон имени
PREFIX = "Грузия"

# Получаем список файлов .jpg
files = [f for f in os.listdir(PHOTO_DIR) if f.endswith('.jpg')]

# Сортируем по дате создания
files.sort(key=lambda f: os.path.getmtime(os.path.join(PHOTO_DIR, f)))

# Переименовываем
day = 1
counter = 1
prev_date = None

for file in files:
    filepath = os.path.join(PHOTO_DIR, file)
    file_date = datetime.fromtimestamp(os.path.getmtime(filepath)).date()

    if prev_date and file_date != prev_date:
        day += 1
        counter = 1

    new_name = f"{PREFIX}-день{day}-{counter:03d}.jpg"
    print(f"  {file} → {new_name}")

    os.rename(filepath, os.path.join(PHOTO_DIR, new_name))
    counter += 1
    prev_date = file_date

print(f"\nГотово! Переименовано {len(files)} файлов.")

Промпт-шаблоны для повторяющихся задач

Промпт-шаблон — это заранее подготовленный запрос к ИИ, который вы используете снова и снова, подставляя новые данные. Храните шаблоны в .md файлах прямо в проекте.

Пакетная обработка файлов

Одно из самых мощных применений автоматизации — обработка множества файлов за раз. Вместо того чтобы работать с каждым файлом по отдельности, вы даёте одну команду.

Сценарий 1: Генерация серии постов

Промпт в Composer 
У меня есть файл topics.md со списком из 10 тем
для блога. Для каждой темы создай отдельный файл
в папке posts/:

posts/01-тема.md
posts/02-тема.md
...

Каждый файл должен содержать:
- Заголовок поста
- Краткое описание (для превью)
- Полный текст поста (300-500 слов)
- 5 хештегов
- Призыв к действию в конце

Создай email-воронку из 5 писем для нового клиента

Терминал в Cursor: базовые команды

В Cursor есть встроенный терминал (открывается через Ctrl+` или Cmd+`). Через него вы запускаете скрипты. Вот самые базовые команды, которые нужно знать:

Команда Что делает Пример
ls Показать список файлов в папке ls photos/
cd Перейти в папку cd my-project/
mkdir Создать папку mkdir notes
cp Скопировать файл cp file.md backup/
mv Переместить/переименовать mv old.md new.md
python script.py Запустить Python-скрипт python rename-photos.py
Совет: Если не знаете, какую команду ввести — просто спросите в Chat: "Как мне запустить скрипт rename-photos.py?" ИИ подскажет точную команду для вашей операционной системы.

Как запустить скрипт пошагово:

1
Попросите ИИ написать скрипт

Опишите задачу в Chat. ИИ создаст файл .py с комментариями и инструкцией.

2
Откройте терминал

Нажмите Ctrl+` (Mac: Cmd+`) — внизу откроется панель терминала.

3
Введите команду запуска

Обычно это python имя_скрипта.py. Нажмите Enter.

4
Если ошибка — скопируйте её в Chat

ИИ объяснит причину и исправит скрипт. Обычно хватает 1-2 итераций.

Примеры автоматизации из реальной жизни

💼 Фрилансер: выставление счетов

Шаблон счёта в Markdown + скрипт, который подставляет данные клиента и сумму. Результат: генерация счёта за 30 секунд вместо 15 минут.

Экономия: ~2 часа в месяц

📰 Блогер: подготовка постов

Промпт-шаблон для генерации постов + пакетная генерация 30 постов на месяц. Ручная доработка каждого занимает 5 минут вместо 30.

Экономия: ~12 часов в месяц

🎓 Преподаватель: проверка работ

Скрипт, который читает работы студентов из папки и генерирует обратную связь по заданным критериям. Преподаватель только проверяет и корректирует.

Экономия: ~5 часов в месяц

📈 Маркетолог: еженедельный отчёт

Шаблон обзора + ИИ анализирует данные проекта и генерирует отчёт. Маркетолог тратит 10 минут на проверку вместо 2 часов на написание.

Экономия: ~7 часов в месяц
Как построить свою систему автоматизации

Подходите к автоматизации системно. Вот пошаговый план:

1
Определите повторяющиеся задачи

На протяжении недели записывайте всё, что вы делаете больше одного раза. Это кандидаты на автоматизацию.

2
Оцените потенциальную экономию

Сколько времени каждая задача занимает сейчас? Сколько будет занимать с автоматизацией? Начните с самых "дорогих" по времени.

3
Создайте шаблоны и скрипты

Для каждой задачи создайте промпт-шаблон или попросите ИИ написать скрипт. Храните всё в папке automation/.

4
Улучшайте итеративно

После каждого использования дорабатывайте шаблон. Через месяц ваша система автоматизации будет работать идеально.
