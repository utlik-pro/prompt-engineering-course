Практика: от задачи к результату

Применяем все инструменты на реальных задачах. Генерируем изображения, видео и подкасты через цепочки промптов. Каждое задание — конкретный use case.

Заметка для преподавателя

Формат: Студенты работают самостоятельно или в парах. 7-10 минут на задание. После каждого — обсуждение лучших результатов.

Ключевое правило: Студенты ВСЕГДА начинают с текстовой LLM для генерации промпта. Не пишут промпты вручную!

Задание 1: Слайд для презентации (Gemini)

Бизнес

Создайте слайд-обложку для бизнес-презентации

Задача:

Вы готовите презентацию для клиента. Тема на выбор:

  • "Цифровая трансформация 2026"
  • "AI-стратегия для ритейла"
  • "Финансовый отчёт Q1"

Инструмент: Gemini Imagen (нужен текст на слайде)

Алгоритм:
  1. Напишите в текстовую LLM: "Напиши промпт для Gemini — слайд-обложка презентации на тему [ваша тема], корпоративный стиль, тёмный фон"
  2. Получите промпт от LLM
  3. Вставьте в Gemini
  4. Оцените результат, попросите LLM доработать если нужно
Пример промпта:
Слайд для бизнес-презентации, заголовок "Цифровая трансформация 2026" крупным белым шрифтом sans-serif, подзаголовок "Стратегия и план действий", тёмно-синий фон с абстрактными световыми линиями, минималистичный корпоративный дизайн, формат 16:9

Задание 2: Фото команды для сайта (Grok Aurora)

HR

Создайте портреты команды для раздела "О нас"

Задача:

Выберите индустрию и создайте 3 портрета сотрудников в едином стиле:

  • Юридическая фирма
  • IT-компания
  • Медицинская клиника
  • Ресторан

Инструмент: Grok Aurora

Алгоритм:
  1. Напишите в LLM: "Напиши 3 промпта для Grok Aurora — портреты сотрудников [индустрия] для сайта. Единый стиль, разные люди."
  2. LLM сгенерирует 3 промпта с разными людьми но в едином стиле
  3. Сгенерируйте все 3 в Grok Aurora
  4. Проверьте единообразие
Совет:

Добавляйте "consistent style", "same lighting setup", "matching color palette" чтобы портреты выглядели как серия

Задание 3: Контент для соцсетей (Nanabanana)

SMM

Создайте серию иллюстраций для Instagram

Задача:

Вы SMM-менеджер (выберите бренд):

  • Кофейня
  • Онлайн-школа
  • Фитнес-клуб
  • Книжный магазин

Нужно 3 поста в едином стиле. Инструмент: Nanabanana

Алгоритм:
  1. Попросите LLM: "Напиши 3 промпта для Nanabanana — серия иллюстраций для Instagram [бренд]. Единый стиль, яркие цвета, digital art."
  2. Сгенерируйте все 3
  3. Оцените: выглядят ли они как серия?

Задание 4: Рекламный видеоролик (Veo3)

Видео

Создайте 8-секундный промо-ролик

Задача:

Выберите бренд из предыдущих заданий и создайте короткий ролик для Stories/Reels.

Инструмент: Grok (Veo3)

Цепочка промптов:
Запрос к LLM:
Напиши промпт для Veo3 — промо-видео кофейни, тёплое, уютное, покажи процесс приготовления латте, кинематографические термины на английском
LLM генерирует:
Cinematic close-up of hands pouring steamed milk into a ceramic cup, creating intricate latte art. Warm morning light through cafe windows, shallow depth of field, camera slowly pulls back revealing cozy interior with exposed brick. Commercial style, warm color grading, inviting atmosphere.

Задание 5: Длинный ролик с диалогом (Sora 2)

Видео

Создайте 20-секундный ролик с диалогом

Задача:

Тот же бренд, что в задании 4. Теперь создайте длинный ролик с диалогом в Sora 2.

Инструмент: Sora 2 (sora.com)

Алгоритм:
  1. Напишите в LLM: "Напиши промпт для Sora 2 — 20-секундный ролик [бренд], добавь 1-2 реплики диалога, описание камеры, освещения и звука. На английском."
  2. Вставьте в sora.com
  3. Сравните с результатом Veo3 из задания 4
Обсудите:
  • Чем отличается результат от Veo3?
  • Как звучит диалог? Естественно?
  • Какой ролик вы бы использовали и для чего?

Задание 6: AI-подкаст (NotebookLM)

Контент

Превратите документ в подкаст

Задача:

Возьмите любой документ и превратите в AI-подкаст:

  • Статья или пост из блога
  • Конспект предыдущего модуля
  • Описание продукта или услуги
  • Свой текст (1-2 страницы)

Инструмент: NotebookLM (notebooklm.google.com)

Алгоритм:
  1. Откройте notebooklm.google.com
  2. Создайте новый notebook
  3. Загрузите документ (PDF, текст или ссылку)
  4. Нажмите "Audio Overview" → "Generate"
  5. Послушайте результат (2-3 мин генерации)
Обсудите:
  • Образование: конспект → подкаст для студентов
  • Маркетинг: блог-пост → аудио для Telegram
  • Юристы: закон → аудио-объяснение для клиента
  • HR: политика компании → аудиогид для новичков

Задание 7: Сравнение платформ

Анализ

Один промпт — три платформы

Задача:

Возьмите один промпт и сгенерируйте изображение в трёх платформах:

  1. Grok Aurora
  2. Gemini Imagen
  3. Nanabanana
Тестовый промпт:
A modern coworking space with large windows, a person working at a laptop, indoor plants, warm afternoon sunlight, minimalist Scandinavian design, high quality
Анализ результатов:
  • Какая платформа дала самый реалистичный результат?
  • Какая лучше передала атмосферу?
  • В чём главные различия?
  • Для какой задачи вы бы выбрали каждый результат?
Памятка: какой инструмент для какой задачи

Фотопортрет, продуктовое фото Grok Aurora
Слайд, баннер, постер с текстом Gemini Imagen
Иллюстрация, арт, креатив Nanabanana
Короткий атмосферный клип (5-8 сек) Veo3 (через Grok)
Длинный ролик с диалогом (до 25 сек) Sora 2
