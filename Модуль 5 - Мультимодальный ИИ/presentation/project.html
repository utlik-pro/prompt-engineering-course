Домашнее задание

Создайте полный мультимедийный контент-пакет для выбранного бизнеса, используя все изученные инструменты. Главное — документ с промптами и описанием цепочек.

Фитнес-клуб "FORCE"

Что нужно создать

Задание 1

3 портрета для страницы "Команда" (Grok Aurora)

Требования:
  • Профессиональные портреты в едином стиле
  • Разные люди (возраст, пол)
  • Соответствуют атмосфере выбранного бизнеса
  • Промпты сгенерированы через текстовую LLM
Рекомендуемый инструмент:

Grok Aurora

Задание 2

2 слайда для презентации (Gemini Imagen)

Требования:
  • Слайд-обложка с названием бизнеса
  • Слайд с ключевой информацией (услуги/преимущества)
  • Читаемый текст, корпоративный стиль
  • Формат 16:9
Рекомендуемый инструмент:

Gemini Imagen

Задание 3

3 поста для Instagram/Telegram (Nanabanana или Gemini)

Требования:
  • Квадратный формат (1:1)
  • Разные типы: продукт, анонс, мотивация/цитата
  • Единый визуальный стиль
Рекомендуемый инструмент:

Nanabanana (иллюстрации) или Gemini (если нужен текст)

Задание 4

Промо-видео 8 секунд (Veo3)

Требования:
  • Короткий рекламный ролик
  • Показывает атмосферу и ценности бизнеса
  • Движение камеры и динамика
  • Промпт написан через текстовую LLM
Рекомендуемый инструмент:

Veo3 (через Grok)

Задание 5

AI-подкаст о бизнесе (NotebookLM)

Требования:
  • Напишите краткое описание бизнеса (1-2 страницы): история, услуги, ценности, команда
  • Загрузите в NotebookLM
  • Сгенерируйте Audio Overview
  • Результат: подкаст-знакомство с бизнесом для клиентов или сотрудников
Рекомендуемый инструмент:

NotebookLM

Задание 6

Документ с промптами (обязательно!)

Требования:
  • Для КАЖДОГО созданного материала: финальный промпт + описание как вы его получили
  • Покажите цепочку: ваш запрос к LLM -> промпт от LLM -> результат
  • Укажите какой инструмент использовали и почему
  • Минимум 3 примера цепочки "промпт для промпта"
Чеклист сдачи проекта
  • 3 портрета команды (Grok Aurora)
  • 2 слайда презентации (Gemini)
  • 3 поста для соцсетей (Nanabanana/Gemini)
  • 1 промо-видео (Veo3)
  • 1 AI-подкаст (NotebookLM)
  • Документ с промптами и цепочками
Критерии оценки
Критерий Баллы Описание
Качество визуалов 20% Разрешение, детализация, отсутствие артефактов
Соответствие бренду 20% Единый стиль, передача ценностей бизнеса
Качество промптов 25% Детализация, использование цепочек "промпт для промпта"
Выбор инструментов 15% Правильный инструмент для каждой задачи
Креативность 10% Оригинальность решений
Документация 10% Полнота документа с промптами
Советы для успешной сдачи:
  • Всегда начинайте с текстовой LLM — не пишите промпты для генераторов вручную
  • Сохраняйте ВСЕ промпты — и запросы к LLM, и финальные промпты для генераторов
  • Делайте несколько вариантов и выбирайте лучший
  • Следите за единообразием стиля во всех материалах
  • В документе покажите минимум 3 полные цепочки "задача -> запрос к LLM -> промпт -> результат"
Заметка для преподавателя

Время на выполнение: 3-4 часа (домашнее задание на неделю)

Формат сдачи: папка с изображениями/видео + документ с промптами

Обратная связь: проведите разбор лучших работ на следующем занятии

Акцент при оценке: на качество цепочек промптов, не только на красивую картинку
