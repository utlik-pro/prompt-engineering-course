# Ресурсы: Отраслевое применение и Knowledge Bases

## AI-платформы для работы

### Основные платформы

#### ChatGPT (OpenAI)
**Ссылка:** https://chat.openai.com

**Описание:**
Универсальная платформа для генерации контента, анализа и творческих задач.

**Лучше всего для:**
- Генерация маркетингового контента
- Творческие задачи
- Быстрые ответы
- Custom GPTs с базами знаний

**Особенности:**
- Custom GPTs для специализации
- Загрузка файлов
- DALL-E для изображений
- Плагины

---

#### Claude (Anthropic)
**Ссылка:** https://claude.ai

**Описание:**
Платформа с фокусом на анализ, длинные документы и структурированные ответы.

**Лучше всего для:**
- Анализ документов
- Структурирование информации
- Code review
- Длинные тексты

**Особенности:**
- Projects с базой знаний
- Длинный контекст (200K tokens)
- Artifacts для кода и документов

---

#### Google Gemini
**Ссылка:** https://gemini.google.com

**Описание:**
Мультимодальная платформа с интеграцией Google-сервисов.

**Лучше всего для:**
- Работа с изображениями
- Интеграция с Google Workspace
- Исследования
- Мультимодальные задачи

---

#### Perplexity AI
**Ссылка:** https://perplexity.ai

**Описание:**
AI-поисковик с цитированием источников.

**Лучше всего для:**
- Исследование отрасли
- Актуальная информация
- Проверка фактов
- Поиск ресурсов

## Инструменты для Knowledge Base

### Notion
**Ссылка:** https://notion.so

**Описание:**
Универсальный инструмент для создания командных KB.

**Преимущества:**
- Гибкая структура
- Базы данных
- Командная работа
- Notion AI интеграция

**Цена:** Бесплатный / от $8/мес

**Ресурсы:**
- Шаблоны: https://notion.so/templates
- Документация: https://notion.so/help

---

### Obsidian
**Ссылка:** https://obsidian.md

**Описание:**
Локальное приложение для персональных KB на основе Markdown.

**Преимущества:**
- Локальное хранение
- Граф связей
- Плагины
- Markdown-based

**Цена:** Бесплатный (sync платный)

**Ресурсы:**
- Плагины: https://obsidian.md/plugins
- Сообщество: https://forum.obsidian.md

---

### Roam Research
**Ссылка:** https://roamresearch.com

**Описание:**
Инструмент для networked thinking и связанных заметок.

**Преимущества:**
- Bidirectional links
- Daily notes
- Outliner формат

**Цена:** от $15/мес

---

### Confluence
**Ссылка:** https://www.atlassian.com/software/confluence

**Описание:**
Enterprise-решение для корпоративных KB.

**Преимущества:**
- Интеграция с Jira
- Командная работа
- Enterprise-функции

**Цена:** от $5.75/пользователь/мес

---

### GitBook
**Ссылка:** https://gitbook.com

**Описание:**
Платформа для документации и публичных KB.

**Преимущества:**
- Красивый дизайн
- Git интеграция
- Публичные страницы

**Цена:** Бесплатный / от $6.70/мес

---

## Инструменты для RAG и AI-интеграции

### LangChain
**Ссылка:** https://langchain.com

**Описание:**
Фреймворк для создания LLM-приложений с RAG.

**Для:** Разработчиков
- Python/JavaScript
- RAG системы
- AI агенты

**Документация:** https://python.langchain.com/docs/

---

### LlamaIndex
**Ссылка:** https://www.llamaindex.ai

**Описание:**
Data framework для LLM-приложений.

**Для:** Разработчиков
- Индексация данных
- RAG
- Query engines

---

### Pinecone
**Ссылка:** https://pinecone.io

**Описание:**
Векторная база данных для AI-приложений.

**Для:** RAG систем
- Vector search
- Высокая производительность
- Простая интеграция

---

### Chroma
**Ссылка:** https://www.trychroma.com

**Описание:**
Open-source векторная БД.

**Для:** Локальных RAG
- Бесплатный
- Простой старт
- Python интеграция
