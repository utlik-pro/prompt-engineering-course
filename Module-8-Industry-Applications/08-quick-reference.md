# Шпаргалка: Отраслевое применение AI

## Быстрая справка по отраслям

### Маркетинг

**Топ-задачи для AI:**
- Генерация контент-планов
- Написание постов и статей
- Email-рассылки
- Рекламные тексты
- SEO-оптимизация

**Базовый промпт:**
```
Ты - маркетолог в сфере [НИША].
Создай [ТИП КОНТЕНТА] для [АУДИТОРИЯ].
Тон: [ДРУЖЕЛЮБНЫЙ/ЭКСПЕРТНЫЙ].
Цель: [КОНВЕРСИЯ/ОСВЕДОМЛЕННОСТЬ].
Формат: [ОПИСАНИЕ ФОРМАТА].
```

**Лучшие инструменты:**
- ChatGPT (контент)
- Jasper (копирайтинг)
- Canva (дизайн)

---

### Образование

**Топ-задачи для AI:**
- Планы уроков
- Тесты и задания
- Адаптация материалов
- Обратная связь
- Презентации

**Базовый промпт:**
```
Ты - методист по [ПРЕДМЕТ].
Создай [ТИП МАТЕРИАЛА] для [АУДИТОРИЯ].
Тема: [ТЕМА].
Продолжительность: [ВРЕМЯ].
Уровень: [БАЗОВЫЙ/СРЕДНИЙ/ПРОДВИНУТЫЙ].
```

**Лучшие инструменты:**
- ChatGPT (материалы)
- Gamma (презентации)
- Kahoot (тесты)

**Топ-задачи для AI:**
- Написание кода
- Code review
- Документация

- Debugging

**Базовый промпт:**

## Формула универсального промпта

```
ПРОМПТ = РОЛЬ + КОНТЕКСТ + ЗАДАЧА + ФОРМАТ + ОГРАНИЧЕНИЯ
```

**Пример:**
```
[РОЛЬ] Ты - senior маркетолог с 10-летним опытом в e-commerce.

[КОНТЕКСТ] Работаешь над запуском нового продукта -
умные часы для спортсменов. Целевая аудитория:
мужчины 25-40 лет, занимающиеся бегом и фитнесом.

[ЗАДАЧА] Создай контент-план на 2 недели для Instagram.

[ФОРМАТ] Таблица: дата, тип контента, тема, CTA, хештеги.

[ОГРАНИЧЕНИЯ] Бюджет на production минимальный,
используем UGC и простые форматы.
```

## Частые ошибки

### Маркетинг:
- Слишком общие промпты без ЦА
- Не указан тон коммуникации
- Публикация без редактуры

**Исправление:** Всегда указывайте ЦА, тон и проверяйте результат.

### Образование:
- Не адаптировано под уровень
- Нет интерактивных элементов
- Не проверены факты

**Исправление:** Указывайте точный уровень, добавляйте практику.

### Программирование:
- Не указана версия языка
- Нет обработки ошибок
- Код не протестирован

**Исправление:** Указывайте версию, просите тесты, всегда проверяйте.

### Аналитика:
- Нет контекста данных
- Слишком общие вопросы
- Не валидируются выводы

**Исправление:** Описывайте контекст, задавайте конкретные вопросы.

## Быстрые промпты по задачам

### Генерация идей:
```
Предложи 10 идей для [ТЕМА].
Контекст: [КОНТЕКСТ].
Критерии: [КРИТЕРИИ].
```

### Анализ текста:
```
Проанализируй текст:
[ТЕКСТ]
Выдели: ключевые тезисы, тон, целевую аудиторию.
```

### Создание структуры:
```
Создай структуру для [ТИП ДОКУМЕНТА].
Тема: [ТЕМА].
Объем: [ОБЪЕМ].
```

### Редактирование:
```
Отредактируй текст:
[ТЕКСТ]
Улучши: читаемость, структуру, ясность.
Сохрани: тон, ключевые идеи.
```

### Перевод формата:
```
Преобразуй [ИСХОДНЫЙ ФОРМАТ] в [ЦЕЛЕВОЙ ФОРМАТ]:
[КОНТЕНТ]
```
