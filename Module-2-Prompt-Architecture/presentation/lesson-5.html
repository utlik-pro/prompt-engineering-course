Урок 5 — Практика

Практика шаблонов

Создаём библиотеку промптов для реальных бизнес-задач

В этом уроке вы создадите 5 профессиональных шаблонов промптов, используя фреймворки и техники из предыдущих уроков.

Заметки для преподавателя

Формат: Практический воркшоп. Каждый студент создаёт шаблоны под свою профессию/отрасль. Время: ~8 минут на задание.

Задание 1: Шаблон для HR

Генератор описаний вакансий

Задача: Создайте шаблон для генерации описаний вакансий.

Переменные для включения:
  • Название должности
  • Отдел/команда
  • Уровень (junior/middle/senior)
  • Формат работы (офис/удалёнка/гибрид)
  • Ключевые навыки (список)
Показать пример решения
job_description.template
# Шаблон: Описание вакансии

[ROLE] Ты — HR-специалист с опытом в IT-рекрутинге.

[INSTRUCTIONS]
Напиши привлекательное описание вакансии:
- Должность: {{position}}
- Отдел: {{department}}
- Уровень: {{level}}
- Формат: {{work_format}}

[REQUIREMENTS]
Ключевые навыки: {{skills}}

[FORMAT]
1. Заголовок с эмодзи
2. О компании (2 предложения)
3. Что предстоит делать (5-7 пунктов)
4. Требования (обязательные + желательные)
5. Мы предлагаем (5-6 пунктов)
6. CTA для отклика

[TONE] Дружелюбный, но профессиональный

Задание 2: Шаблон для маркетинга

Генератор рекламных текстов

Задача: Создайте универсальный шаблон для рекламных объявлений.

Переменные: продукт, УТП, целевая аудитория, платформа (Google/Facebook/Instagram), бюджет тона (агрессивный/мягкий)
Показать пример решения
ad_copy.template
# Шаблон: Рекламный текст (CRISPE)

[CAPACITY & ROLE]
Ты — performance-маркетолог с опытом в {{platform}}.

[INSIGHT]
Продукт: {{product}}
УТП: {{utp}}
Целевая аудитория: {{audience}}

[STATEMENT]
Создай 3 варианта рекламного объявления для {{platform}}.

[PERSONALITY]
Тон: {{tone}}

[EXPERIMENT]
Для каждого варианта используй разный подход:
1. Фокус на боли клиента
2. Фокус на выгоде
3. Социальное доказательство

Формат для {{platform}}:
- Заголовок: до {{title_limit}} символов
- Описание: до {{desc_limit}} символов
- CTA: 2-3 слова

Задание 3: Шаблон для продаж

Генератор коммерческих предложений

Задача: Создайте шаблон для персонализированных КП.

Показать пример решения
commercial_proposal.template
# Шаблон: Коммерческое предложение (RISEN)

[ROLE]
Ты — опытный sales-менеджер в сфере {{industry}}.

[INSTRUCTIONS]
Напиши персонализированное КП для:
- Компания: {{client_company}}
- Контакт: {{contact_name}}, {{contact_position}}
- Их проблема: {{pain_point}}

[STEPS]
1. Персонализированное приветствие
2. Демонстрация понимания их проблемы
3. Наше решение ({{our_product}})
4. Конкретные выгоды (3-5 пунктов с цифрами)
5. Кейс похожего клиента
6. Условия и цены
7. Следующий шаг (CTA)

[END GOAL]
КП, которое приведёт к встрече.

[NARROWING]
- Длина: 1 страница
- Без технического жаргона
- Фокус на ROI

Задание 4: Шаблон для аналитики

Анализатор данных

Задача: Создайте шаблон для анализа данных с JSON-выводом.

Показать пример решения
data_analysis.template
# Шаблон: Анализ данных

Проанализируй следующие данные:

{{data}}

Верни результат в JSON:
{
  "summary": "краткое резюме",
  "key_metrics": {
    "{{metric_1}}": значение,
    "{{metric_2}}": значение
  },
  "trends": ["тренд 1", "тренд 2"],
  "anomalies": ["аномалия, если есть"],
  "recommendations": ["рекомендация 1", "рекомендация 2"],
  "confidence": число от 0 до 1
}

Отвечай ТОЛЬКО валидным JSON.

Задание 5: Ваш собственный шаблон

Свободное задание

Задача: Создайте шаблон для задачи из вашей профессиональной области.

Требования:
  • Используйте один из фреймворков (RISEN, CRISPE, CREATE)
  • Минимум 5 переменных
  • Укажите формат вывода
  • Добавьте документацию к шаблону

Чек-лист практики

  • Задание 1: HR-шаблон создан
  • Задание 2: Маркетинг-шаблон создан
  • Задание 3: Продажи-шаблон создан
  • Задание 4: Аналитика-шаблон создан
