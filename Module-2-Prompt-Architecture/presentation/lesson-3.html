Создание переиспользуемых шаблонов с переменными и плейсхолдерами

Что такое промпт-шаблоны?

Промпт-шаблон — это заготовка промпта с переменными (плейсхолдерами), которые заполняются конкретными данными перед отправкой.
Шаблон: 
Напиши {{тип_текста}} для {{компания}}
о {{тема}}. Длина: {{длина}} слов.
Тон: {{тон}}.
Заполненный промпт: 
Напиши пресс-релиз для Apple
о новом iPhone 16. Длина: 300 слов.
Тон: профессиональный.

Синтаксис переменных

Популярные форматы плейсхолдеров
Формат Пример Где используется
{{переменная}} {{company_name}} Mustache, Handlebars
{переменная} {company_name} Python f-strings
[ПЕРЕМЕННАЯ] [COMPANY_NAME] Простой текст
<переменная> <company_name> XML-стиль
Рекомендация: Используйте {{двойные_скобки}} — они хорошо выделяются и редко встречаются в обычном тексте.

Примеры шаблонов

Шаблон для email-рассылки
email_campaign.template 
# Шаблон: Email-рассылка

## Переменные:
- {{company}} — название компании
- {{product}} — продукт/услуга
- {{discount}} — размер скидки
- {{deadline}} — дедлайн акции
- {{audience}} — целевая аудитория
- {{tone}} — тон (дружелюбный/профессиональный/срочный)

## Промпт:

Ты — email-маркетолог компании {{company}}.

Напиши продающее письмо для email-рассылки:
- Продукт: {{product}}
- Скидка: {{discount}}
- Акция действует до: {{deadline}}
- Целевая аудитория: {{audience}}

Формат:
1. Тема письма (до 50 символов)
2. Прехедер (до 90 символов)
3. Основной текст (150-200 слов)
4. CTA-кнопка (2-4 слова)

Тон: {{tone}}
Пример заполнения:
  • company: TechShop
  • product: наушники Sony WH-1000XM5
  • discount: 25%
  • deadline: 31 декабря
  • audience: любители музыки 25-45 лет
  • tone: дружелюбный
Шаблон для анализа конкурентов
competitor_analysis.template 
# Шаблон: Анализ конкурентов

## Переменные:
- {{my_company}} — моя компания
- {{industry}} — отрасль
- {{competitors}} — список конкурентов
- {{focus_areas}} — на что обратить внимание
- {{output_format}} — формат вывода

## Промпт:

Ты — бизнес-аналитик с опытом в {{industry}}.

Проведи конкурентный анализ для компании {{my_company}}.

Конкуренты для анализа:
{{competitors}}

Особое внимание удели:
{{focus_areas}}

Структура отчёта:
1. Обзор рынка (2-3 предложения)
2. Таблица сравнения конкурентов
3. Сильные стороны каждого конкурента
4. Слабые стороны и возможности
5. Рекомендации для {{my_company}}

Формат вывода: {{output_format}}
Шаблон для генерации контента
content_generator.template 
# Шаблон: Генератор контента

## Переменные:
- {{content_type}} — тип (пост/статья/скрипт)
- {{platform}} — платформа (Instagram/LinkedIn/YouTube)
- {{topic}} — тема
- {{keywords}} — ключевые слова (опционально)
- {{length}} — длина
- {{cta}} — призыв к действию

## Промпт:

Создай {{content_type}} для {{platform}}.

Тема: {{topic}}

{{#if keywords}}
Включи ключевые слова: {{keywords}}
{{/if}}

Требования:
- Длина: {{length}}
- Призыв к действию: {{cta}}
- Добавь релевантные эмодзи
- Включи 3-5 хэштегов

Стиль: соответствующий {{platform}}

Условная логика в шаблонах

Иногда нужно включать или исключать части промпта в зависимости от условий:

Условные блоки 
Напиши описание товара для {{product_name}}.

{{#if has_discount}}
ВАЖНО: Товар со скидкой {{discount}}%!
Подчеркни выгоду покупки сейчас.
{{/if}}

{{#if is_new}}
Это НОВИНКА! Сделай акцент на инновационности.
{{/if}}

{{#if has_reviews}}
Упомяни, что товар имеет рейтинг {{rating}}
на основе {{review_count}} отзывов.
{{else}}
Не упоминай отзывы.
{{/if}}

Целевая аудитория: {{audience}}
Совет: Даже если вы не используете программный шаблонизатор, можно использовать эту логику вручную — просто удаляйте ненужные блоки перед отправкой.
