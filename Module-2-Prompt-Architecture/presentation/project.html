Корпоративная библиотека промптов

Создание профессиональной коллекции шаблонов для бизнеса

Описание проекта

Создайте библиотеку из 8 профессиональных промптов для компании, работающей в выбранной вами отрасли. Каждый промпт должен:

  • Использовать один из изученных фреймворков (RISEN, CRISPE или CREATE)
  • Запрашивать структурированный вывод (JSON, таблица или форматированный список)
  • Быть шаблоном с переменными для повторного использования
  • Включать документацию по использованию

Требования к библиотеке

8 обязательных промптов по категориям
1. HR / Управление персоналом

Подбор кандидатов, составление описания вакансий, оценка резюме

2. Маркетинг

Создание контента, анализ аудитории, планирование кампаний

3. Продажи

Коммерческие предложения, скрипты звонков, работа с возражениями

4. Клиентский сервис

Ответы на жалобы, FAQ, шаблоны коммуникации

5. Аналитика

Анализ данных, отчёты, прогнозирование

6. Операционная деятельность

Процедуры, чек-листы, инструкции

7. Стратегия / Планирование

SWOT-анализ, планы развития, бизнес-модели

8. Обучение / Развитие

Учебные материалы, тесты, программы onboarding

Документация промпта 
## Анализатор резюме кандидатов
**Категория:** HR / Подбор персонала
**Фреймворк:** RISEN
**Формат вывода:** JSON

### Назначение
Автоматизированный анализ резюме кандидата с оценкой
соответствия требованиям вакансии и рекомендациями по собеседованию.

### Переменные
- `{{job_title}}` — название вакансии
- `{{requirements}}` — ключевые требования к кандидату
- `{{resume_text}}` — текст резюме кандидата
- `{{company_values}}` — ценности компании (опционально)

### Промпт
```
**Role:** Ты опытный HR-специалист с 10-летним стажем в IT-рекрутинге.

**Instructions:** Проанализируй резюме кандидата на позицию {{job_title}}.

**Steps:**
1. Оцени соответствие hard skills требованиям
2. Проанализируй опыт работы и достижения
3. Выяви потенциальные риски (gaps, частая смена работы)
4. Сформулируй вопросы для собеседования

**End goal:** Предоставь структурированную оценку кандидата.

**Narrowing:** Верни ответ в JSON-формате:
{
  "candidate_name": "имя из резюме",
  "match_score": 0-100,
  "skills_analysis": {
    "matched": ["навык1", "навык2"],
    "missing": ["навык3"],
    "bonus": ["навык4"]
  },
  "experience_summary": "краткий анализ опыта",
  "red_flags": ["риск1"] или [],
  "interview_questions": ["вопрос1", "вопрос2", "вопрос3"],
  "recommendation": "hire/maybe/reject",
  "reasoning": "обоснование рекомендации"
}

**Требования к вакансии:**
{{requirements}}

**Резюме кандидата:**
{{resume_text}}
```

### Пример использования
**Входные данные:**
- job_title: "Senior Python Developer"
- requirements: "Python 5+ лет, Django, PostgreSQL, опыт с микросервисами"
- resume_text: "[текст резюме]"

**Ожидаемый результат:**
```json
{
  "candidate_name": "Иван Петров",
  "match_score": 85,
  "skills_analysis": {
    "matched": ["Python 6 лет", "Django", "PostgreSQL"],
    "missing": ["микросервисы - опыт не указан явно"],
    "bonus": ["FastAPI", "Docker", "AWS"]
  },
  "experience_summary": "Сильный бэкенд-разработчик с опытом в финтех...",
  "red_flags": [],
  "interview_questions": [
    "Расскажите о вашем опыте работы с микросервисной архитектурой",
    "Какой самый сложный проект на Django вы реализовали?",
    "Как вы оптимизировали производительность PostgreSQL?"
  ],
  "recommendation": "hire",
  "reasoning": "Кандидат превышает требования по основным навыкам..."
}
```

### Советы по использованию
- Копируйте резюме без форматирования (plain text)
- Добавляйте ценности компании для оценки cultural fit
- Используйте результат как основу, а не финальное решение

## Анализатор конкурентного контента
**Категория:** Маркетинг / Конкурентный анализ
**Фреймворк:** CRISPE
**Формат вывода:** Markdown с таблицами

### Назначение
Глубокий анализ контент-стратегии конкурента с выявлением
сильных сторон и возможностей для дифференциации.

### Переменные
- `{{competitor_name}}` — название конкурента
- `{{competitor_content}}` — примеры контента конкурента
- `{{our_brand}}` — название нашего бренда
- `{{our_audience}}` — описание нашей целевой аудитории

### Промпт
```
**Capacity:** Ты — senior контент-стратег с опытом работы
в топовых digital-агентствах. Специализация: конкурентный
анализ и контент-маркетинг в B2B.

**Role:** Проведи экспертный анализ контент-стратегии конкурента.

**Insight:** Наша компания {{our_brand}} конкурирует за аудиторию
{{our_audience}}. Нам важно понять, что работает у конкурента,
и найти незанятые ниши.

**Statement:** Проанализируй контент {{competitor_name}} и дай
рекомендации по дифференциации нашей стратегии.

**Personality:** Будь аналитичным, но практичным. Избегай общих
фраз — давай конкретные, применимые рекомендации.

**Experiment:** Представь результат в следующем формате:

## Анализ контента {{competitor_name}}

### Ключевые метрики
| Параметр | Оценка | Комментарий |
|----------|--------|-------------|
| Частота публикаций | X/неделю | ... |
| Основные форматы | ... | ... |
| Tone of voice | ... | ... |
| Вовлечённость | ... | ... |

### Сильные стороны (что копировать)
1. ...
2. ...
3. ...

### Слабые стороны (наши возможности)
1. ...
2. ...
3. ...

### Рекомендации для {{our_brand}}
1. **[Название тактики]:** Описание + пример реализации
2. ...
3. ...

### Контент-идеи для дифференциации
- Идея 1
- Идея 2
- Идея 3

**Контент конкурента для анализа:**
{{competitor_content}}
```

### Советы по использованию
- Собирайте 5-10 примеров контента конкурента
- Включайте посты с высоким engagement
- Обновляйте анализ раз в квартал
