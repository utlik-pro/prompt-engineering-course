Режим студента

Обучение на примерах

Научитесь обучать AI на примерах вместо инструкций.
Few-shot prompting — одна из самых мощных техник.

Заметка для преподавателя

Общая продолжительность модуля: ~3 часа (6 уроков + домашнее задание)

Рекомендация: 70% времени отводите на практику. Студенты должны создать минимум 5 работающих Few-shot промптов.

Ключевая цель: Студенты понимают разницу между Zero-shot, Few-shot и Many-shot, и умеют подбирать качественные примеры.

Что вы изучите в этом модуле:
  • Zero-shot prompting — без примеров, только инструкция
  • Few-shot prompting — 2-10 примеров для обучения
  • Many-shot prompting — десятки примеров для максимальной точности
  • In-Context Learning — как модель "обучается" в контексте
  • Prompt Ensembling — комбинирование нескольких промптов
  • Structured Outputs — гарантированный JSON без примеров формата
  • Prompt Caching — скидка 75-90% при повторном использовании промптов
  • Мультимодальный Few-shot — примеры с изображениями, аудио, видео

💡 Главная мысль модуля

"Вместо того, чтобы объяснять AI, ЧТО делать словами —
покажите КАК делать на примерах."

0️⃣

Zero-shot Prompting

Просто даете задачу без примеров. Быстро, но менее точно. Подходит для простых, стандартных задач.

Без примеров
🎯

Few-shot Prompting

Показываете 2-10 примеров — модель улавливает паттерн. Точнее, требует подготовки примеров.

2-10 примеров
📚

Many-shot Prompting

Загружаете десятки или сотни примеров. Максимальная точность для специфичных задач.

50+ примеров
🧠

In-Context Learning

Модель "обучается" прямо в контексте промпта, не меняя веса. Как актер, который сразу входит в роль.

Мгновенное обучение
🔀

Prompt Ensembling

Несколько вариаций промпта + комбинирование результатов. Для критически важных задач.

Комбинирование

Новые техники 2026

📋

Structured Outputs

Гарантированный JSON по схеме. Примеры теперь нужны только для стиля и логики, а не для формата.

GPT-5, Claude, Gemini
💰

Prompt Caching

Скидка 75-90% при повторном использовании системных промптов с примерами. Few-shot стал в 4-10x дешевле.

Все провайдеры
