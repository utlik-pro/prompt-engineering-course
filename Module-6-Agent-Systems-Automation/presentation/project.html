Домашнее задание: Создание AI агента

Требования к проекту:

  • Выбрать одну из предложенных тем или свою
  • Создать работающего AI агента
  • Написать детальные инструкции
  • Протестировать и задокументировать результаты
Заметка для преподавателя

Дедлайн: до следующего занятия

Формат сдачи: ссылка на агента + документация

Оценка: по критериям ниже (0-100 баллов)

Описание задания

Создайте полноценного AI агента для реальной бизнес-задачи. Выберите одну из предложенных тем или придумайте свою (согласуйте с преподавателем).

Варианты проектов

1

Клиентский сервис

Агент поддержки клиентов: ответы на FAQ, маршрутизация запросов, эскалация сложных случаев.

Платформа: Custom GPT, Coze

2

Продуктовый менеджер

Агент для разработки продуктов: генерация идей, анализ конкурентов, создание user stories.

Платформа: Claude Project

3

Автоматизация бизнеса

Workflow в n8n: автоматическая обработка заявок, уведомления, интеграция с CRM.

Платформа: n8n + OpenAI

4

Своя тема

Предложите свою идею агента для решения реальной рабочей задачи.

Согласуйте: с преподавателем

Требования к проекту

Минимальные требования (60 баллов)
  • Работающий агент на выбранной платформе
  • Четкие инструкции (роль, функции, принципы)
  • Минимум 3 conversation starters
  • Базовое тестирование (5+ запросов)
Продвинутые требования (+20 баллов)
  • Загруженные файлы в Knowledge (шаблоны, данные)
  • Включенные инструменты (Web Browsing, Code Interpreter)
  • Примеры ответов в инструкциях
  • Ограничения и негативные примеры
Бонусные баллы (+20 баллов)
  • Автоматизация через n8n или API
  • Интеграция с внешними сервисами
  • Детальная документация с примерами
  • Презентация проекта на занятии

Формат сдачи

Документация

Markdown

Примеры успешных проектов

Хороший проект
  • Четкая структура инструкций
  • Релевантные файлы в Knowledge
  • Разнообразные тесты
  • Документация с примерами
Плохой проект
  • Короткие, нечеткие инструкции
  • Нет тестирования
  • Отсутствует документация
  • Не работает на edge cases

Советы по выполнению

1

Начните рано

Не откладывайте на последний день. Итеративное улучшение требует времени.

2

Тестируйте много

Проверяйте агента на разных запросах, включая нестандартные.

3

Документируйте

Записывайте все изменения и результаты тестов по ходу работы.

4

Просите фидбек

Попросите коллег или друзей протестировать агента.

Критерии оценки

Критерий Баллы Описание
Функциональность 0-30 Агент работает, выполняет заявленные функции
Качество инструкций 0-25 Четкость, структура, полнота
Тестирование 0-20 Разнообразие тестов, качество результатов
Документация 0-15 Полнота отчета, примеры, выводы
Креативность 0-10 Оригинальность решения, дополнительные фичи

  • Документация заполнена по шаблону
  • Минимум 5 тестов с результатами
  • Инструкции включены в отчет
  • Сдано до дедлайна
