Домашнее задание: Агент для реальной задачи

Требования к проекту:

  • Создать рабочего агента под реальную задачу из вашей работы
  • Написать детальную инструкцию с ролью, функциями и форматом
  • Протестировать на 5+ разных запросах и задокументировать результаты
  • Поделиться результатом с преподавателем или коллегами
Заметка для преподавателя

Дедлайн: до следующего занятия.

Формат сдачи: ссылка на агента + скриншоты тестирования + краткое описание в свободной форме или по шаблону ниже.

Оценка: по критериям ниже (0–100 баллов). На следующем занятии — 5 минут на демонстрацию лучших работ.

Создайте AI-агента, который реально помогает вам в работе. Не демо, не тест — настоящий инструмент, который вы будете использовать каждый день. Выберите платформу из тех, что изучили в модуле, и тему из своей профессиональной жизни.

Варианты проектов

💼

Профессиональный ассистент

Агент для вашей ежедневной работы: пишет тексты, анализирует данные, готовит документы по вашей профессии.

👥

Командный инструмент

Агент для всей команды. Поделитесь ссылкой с коллегами — пусть они тоже протестируют и дадут обратную связь.

💬

Клиентский сервис

Агент для ответов клиентам. Опубликуйте в Telegram через Coze и дайте ссылку реальным пользователям.

🧠

Личный помощник

Агент для личной продуктивности: планирование, напоминания, рефлексия, обучение.

Требования к проекту

Минимум (60 баллов)
  • Работающий агент на любой из платформ курса
  • Инструкция с ролью, функциями и форматом ответов
  • 5+ тестов с разными запросами
  • Скриншоты результатов тестирования
Хорошо (+20 баллов)
  • Агент доступен по публичной ссылке (Coze или Custom GPT)
  • Протестирован реальным пользователем (не автором)
  • Настроена память или загружены файлы знаний
  • Добавлены плагины или инструменты (Web Search, Code и т.д.)
Отлично (+20 баллов)
  • Агент опубликован в Telegram и доступен пользователям
  • Используется командой или клиентами в реальной работе
  • Итеративно улучшен после первого тестирования
  • Презентация агента на следующем занятии (5 минут)
