Создаём агента в ChatGPT: Custom GPTs

Цели урока:

  • Понять, что такое Custom GPT и зачем он нужен
  • Пошагово создать своего первого агента прямо на занятии
  • Освоить написание инструкций (Instructions) для агента
  • Попробовать готовые примеры и адаптировать под себя
Заметка для преподавателя

Время: 40 минут — 10 мин теория + 25 мин практика + 5 мин обсуждение

Формат: Все создают агента прямо на занятии. Преподаватель демонстрирует на экране, студенты повторяют.

Требования: Нужен ChatGPT аккаунт. Бесплатный аккаунт позволяет просматривать и использовать чужие GPTs. Для создания своих — нужен Plus ($20/мес) или Pro ($200/мес). Если у студентов нет Plus — работают в парах или используют готовые примеры из урока.

Что такое Custom GPT?

Custom GPT — это ваш персональный AI-помощник с вашими инструкциями, знаниями и стилем общения. Работает на платформе ChatGPT, настраивается без единой строчки кода. Вы один раз пишете "правила игры" — и агент работает по ним с любым пользователем.
Обычный ChatGPT
  • Общий помощник — всё подряд
  • Нужно каждый раз объяснять контекст
  • Не знает вашу специфику
  • Универсальный тон ответов
  • Не помнит ваши предпочтения
Custom GPT
  • Специализированный эксперт в вашей нише
  • Всегда знает контекст вашей работы
  • Использует ваши документы и шаблоны
  • Отвечает в нужном вам стиле и формате
  • Можно поделиться с командой или клиентами

Что нужно для начала

  • Аккаунт ChatGPT на chatgpt.com — Plus или Pro для создания Custom GPTs
  • 15 минут времени — этого хватит на создание первого рабочего агента
  • Идея для агента — выберите из примеров ниже или придумайте свою

Создаём агента: пошагово

Шаг 1: Открываем конструктор GPT
  1. Заходим на chatgpt.com и авторизуемся
  2. В левом боковом меню находим раздел "Explore GPTs"
  3. В правом верхнем углу нажимаем кнопку "Create"
  4. Открывается конструктор. Видим две вкладки: "Create" (диалог с ИИ-помощником) и "Configure" (ручная настройка)
  5. Выбираем "Configure" — там полный контроль над всеми параметрами
Совет: Вкладка "Create" — это мастер-помощник, который сам задаёт вопросы. Полезна для первого знакомства. Вкладка "Configure" — для опытных пользователей, даёт полный контроль.
Шаг 2: Называем агента и добавляем описание

Name (Название) — как будет называться ваш агент. Сделайте его понятным и запоминающимся.

  • Хорошо: "Маркетолог для Instagram", "HR-ассистент Иванова и К"
  • Плохо: "GPT1", "Мой помощник"

Description (Описание) — короткое объяснение для чего этот агент. Видно пользователям в магазине GPTs.

  • Пример: "Помогает создавать посты для LinkedIn, Instagram и Telegram. Пишите — получите готовый текст с хэштегами."

Profile picture — аватар агента. Можно попросить ChatGPT сгенерировать прямо здесь, нажав на иконку с карандашом.

Шаг 3: Пишем Instructions — сердце агента

Instructions — это главное поле. Здесь вы описываете "правила" агента: кто он, что делает, как отвечает. Это и есть системный промпт.

Структура хороших Instructions:

Структура блоков
  • Роль: "Ты — опытный маркетолог..."
  • Функции: "ЧЕМ ПОМОГАЮ: ..."
  • Уточняющие вопросы: "ПЕРЕД РАБОТОЙ СПРАШИВАЮ: ..."
  • Ограничения: "НИКОГДА не делаю: ..."
  • Формат ответа: "ФОРМАТ: готовый текст + ..."
Берите готовые примеры из раздела ниже и вставляйте прямо сюда. Потом корректируйте под себя.
Шаг 4: Настраиваем возможности и сохраняем

Capabilities (Возможности):

  • Web Search — агент может искать актуальную информацию в интернете. Включайте для агентов по новостям, рынкам, трендам.
  • DALL-E Image Generation — агент создаёт изображения по описанию.
  • Code Interpreter — агент анализирует таблицы Excel/CSV, строит графики, работает с данными.

Knowledge (База знаний):

  • Загружайте PDF, DOCX, TXT — агент будет отвечать на основе ваших документов
  • Примеры: прайс-лист компании, корпоративный стиль, шаблоны договоров, FAQ

Сохранение: Нажимаем "Save" → выбираем кому доступен агент: "Only me", "Anyone with a link" или "Everyone". Нажимаем "Confirm".

Агент создан! Теперь его можно тестировать прямо в окне справа.

Лайфхаки для сильных GPTs

🎭

Начни с роли

Напиши: "Ты — [профессия] с [опытом] в [области]". Чем конкретнее роль — тем точнее ответы. Не "помощник", а "старший копирайтер с 8 годами в B2B SaaS".

🚫

Укажи чего НЕ делать

Добавь раздел "НИКОГДА:". Например: "Никогда не давай медицинских советов", "Не пиши тексты длиннее 300 слов без запроса", "Не делай выводы без данных".

💡

Добавь примеры ответов

В конце Instructions вставь 1-2 примера диалога в формате "Вопрос: ... / Ответ: ...". Это few-shot learning — агент будет копировать этот стиль.

📁

Загрузи документы

В раздел Knowledge загрузи шаблоны, гайдлайны бренда, прайс-лист, FAQ. Агент будет отвечать на основе ваших реальных документов, а не общих знаний.

  • Структура Instructions: Роль → Функции → Уточняющие вопросы → Ограничения → Формат ответа.
  • Knowledge позволяет загрузить ваши документы — агент будет отвечать на их основе.
  • Capabilities: Web Search для актуальных данных, Code Interpreter для анализа файлов и таблиц.
