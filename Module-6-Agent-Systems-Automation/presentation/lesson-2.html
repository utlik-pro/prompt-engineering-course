ChatGPT Custom GPTs & Claude Projects

Цели урока:

  • Научиться создавать Custom GPTs в ChatGPT
  • Освоить работу с Claude Projects
  • Понять различия между платформами

Время: 45 минут (15 мин Custom GPTs + 15 мин Claude Projects + 15 мин практика)

Демо: Создайте маркетингового агента в реальном времени, показывая каждый шаг.

ChatGPT Custom GPTs

Custom GPTs — это персонализированные версии ChatGPT с собственными инструкциями, знаниями и возможностями. Создаются без программирования. В 2025-2026 ChatGPT также получил Operator — агента, управляющего браузером и приложениями.

Преимущества Custom GPTs

🎯

Простота

Интуитивный интерфейс без кода

🔧

Инструменты

Поиск, Image Gen, Code Interpreter, Operator

📚

Знания

Загрузка файлов и документов

🌐

Публикация

Делитесь с другими пользователями

Пошаговое создание Custom GPT

Шаг 1: Создание нового GPT
  1. Откройте ChatGPT
  2. В левом меню нажмите "Explore GPTs"
  3. Нажмите "Create" в правом верхнем углу
  4. Выберите "Configure" для детальной настройки
Шаг 2: Базовая информация
📝 Пример настройки
Название: Маркетинговый ассистент

Описание: Эксперт по маркетингу, который помогает создавать контент для социальных сетей, планировать кампании и анализировать эффективность.

Conversation starters:
• Создай контент-план на неделю для LinkedIn
• Напиши email-рассылку для привлечения клиентов
• Проанализируй тренды в маркетинге
• Создай рекламное объявление для Facebook
Шаг 3: Инструкции (Instructions)
Instructions 
Ты - опытный маркетолог с 10+ летним стажем в digital-маркетинге.

Твоя задача - помогать создавать эффективный контент и стратегии.

ТВОИ ФУНКЦИИ:
1. Создание контента для социальных сетей (LinkedIn, Facebook, Instagram)
2. Разработка email-кампаний и рассылок
3. Планирование контент-стратегий
4. Анализ трендов и конкурентов
5. Создание рекламных текстов

ПРИНЦИПЫ РАБОТЫ:
- Всегда учитывай целевую аудиторию
- Адаптируй тон и стиль под платформу
- Предлагай конкретные решения
- Используй данные для обоснования
- Следи за актуальными трендами

ФОРМАТ ОТВЕТОВ:
- Структурированные и легко читаемые
- С конкретными примерами
- С практическими советами
- С метриками для измерения
Шаг 4: Настройка возможностей

Capabilities (Возможности):

  • Web Browsing — поиск актуальной информации
  • Code Interpreter — анализ данных, работа с файлами
  • Image Generation — генерация изображений (GPT-4o native)

Knowledge (Знания):

Загрузите файлы с примерами успешных кампаний, шаблонами контента, гайдлайнами бренда.

Claude Projects

Claude Projects — функция для создания долгосрочных проектов с расширенным контекстом до 1M токенов, веб-поиском, генерацией изображений и управлением файлами. Claude поддерживает MCP (Model Context Protocol) для подключения внешних инструментов — GitHub, Notion, базы данных и тысячи других сервисов.

Преимущества Claude Projects

✅ Сильные стороны
  • Долгосрочная память между сессиями
  • Огромный контекст (1M токенов)
  • Работа с большими документами
  • Организация по проектам
📁 Поддерживаемые форматы
  • Текст: .txt, .md, .csv
  • Документы: .pdf, .docx, .pptx
  • Данные: .xlsx, .json, .xml
  • Код: .py, .js, .html, .css

Создание Claude Project

Шаг 1: Создание проекта
  1. Откройте Claude.ai
  2. Нажмите на иконку Projects в левом меню
  3. Нажмите "New Project"
  4. Введите название и описание проекта
Шаг 2: Настройка проекта
📝 Пример настройки
Название: Аналитический ассистент для e-commerce

Описание: Проект для анализа данных интернет-магазина, создания отчетов и рекомендаций по оптимизации продаж.
Шаг 3: Project Instructions
Project Instructions 
Ты - эксперт по аналитике e-commerce с опытом работы в различных интернет-магазинах.

ТВОИ ФУНКЦИИ:
1. Анализ данных о продажах и трафике
2. Создание отчетов и дашбордов
3. Выявление трендов и паттернов
4. Рекомендации по оптимизации
5. Прогнозирование и планирование

ПРИНЦИПЫ РАБОТЫ:
- Используй конкретные данные и метрики
- Предлагай практические рекомендации
- Учитывай специфику e-commerce
- Фокусируйся на ROI и конверсии

ФОРМАТ ОТЧЕТОВ:
- Executive Summary (краткие выводы)
- Ключевые метрики (с визуализацией)
- Анализ трендов (сравнение периодов)
- Рекомендации (приоритизированные)
- План действий (конкретные шаги)
Шаг 4: Загрузка данных

Загрузите файлы в раздел Project Knowledge:

  • sales_data_2024.xlsx — данные о продажах
  • traffic_analytics.csv — аналитика трафика
  • customer_feedback.pdf — отзывы клиентов
  • marketing_campaigns.json — данные о кампаниях

Claude будет использовать эти файлы как контекст для всех ответов в проекте.

Claude Code — агент для разработчиков

Claude Code (2025) — терминальный AI агент от Anthropic, который читает файлы, запускает команды, пишет и тестирует код. Работает прямо в вашем проекте через CLI. Поддерживает MCP для подключения баз данных и внешних инструментов.

Когда использовать?

🎯 Custom GPTs
  • Простые задачи с чёткими инструкциями
  • Публичные агенты для широкой аудитории
  • Быстрое прототипирование
  • Нужны встроенные инструменты (DALL-E, поиск)
📚 Claude Projects + MCP
  • Сложные аналитические задачи
  • Работа с большими объёмами данных
  • Долгосрочные проекты
  • Нужен глубокий контекст
  • Подключение внешних инструментов через MCP

Проверка знаний

Какая платформа лучше подходит для работы с большими документами и долгосрочными проектами?
ChatGPT Custom GPTs
Claude Projects
Обе платформы одинаково подходят
Ни одна из них
📝 Ключевые выводы
  • Custom GPTs — идеальны для быстрого создания публичных агентов с встроенными инструментами
  • Claude Projects — лучше для сложных аналитических задач с большими документами
  • Instructions — ключевой элемент настройки: роль, задачи, принципы, формат
  • Knowledge — загружайте релевантные файлы для улучшения ответов
  • Выбор платформы зависит от конкретных задач и требований проекта
  • MCP превращает Claude в мощный агент с доступом к GitHub, Notion, базам данных и сотням других инструментов
  • ChatGPT Operator (2025) — агент, управляющий браузером и веб-приложениями без участия человека
