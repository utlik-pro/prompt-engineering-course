Создаём агента в Coze — самая мощная бесплатная платформа

Цели урока:

  • Познакомиться с Coze — самой функциональной бесплатной платформой
  • Создать агента с плагинами и долгосрочной памятью
  • Настроить Workflow для автоматизации задач
  • Опубликовать агента в Telegram
Заметка для преподавателя

Время: 45 минут. Coze от ByteDance — бесплатный аналог ChatGPT GPTs с гораздо большими возможностями.

Подготовка: Все регистрируются на coze.com перед началом урока или в самом начале.

Фишка урока: Публикация в Telegram — студенты уходят с готовым AI-ботом в своём Telegram. Это мощный практический результат.

Coze — платформа от ByteDance (создатели TikTok) для создания AI-агентов. Полностью бесплатная. Более мощная, чем Custom GPTs: есть плагины, долгосрочная память, Workflow-автоматизация и публикация в Telegram, Discord, Slack.

Пошаговое создание агента в Coze

Шаг 1: Создаём нового бота
  1. Зайдите на coze.com и войдите в аккаунт
  2. Нажмите "Create Bot"
  3. Введите название и описание агента
  4. Выберите модель: GPT-4o или Claude 3.5 Sonnet
📝 Пример
Bot Name: Мой деловой помощник
Description: Помогает с деловой перепиской, анализом рынка и планированием задач.
Шаг 2: Persona & Prompt — инструкция агента

Вставляем инструкцию в поле Persona & Prompt. Та же структура: роль, функции, принципы.

Пример Prompt 
Ты — умный деловой ассистент.

ЧЕМ ПОМОГАЮ:
- Написание деловых писем и коммерческих предложений
- Анализ рынка и поиск актуальной информации
- Планирование задач и напоминания
- Ответы на вопросы по бизнесу

ПРИНЦИПЫ:
- Отвечай по делу, кратко
- Используй поиск для актуальных данных
- Помни предпочтения пользователя
Шаг 3: Plugins — добавляем суперсилы

В разделе Plugins добавляем инструменты. Для большинства агентов рекомендуем:

  • Web Search — актуальная информация из интернета
  • Image Generation — создание изображений по описанию
  • News — свежие новости по теме
  • Calculator — точные вычисления
Шаг 4: Memory — долгосрочная память

В разделе Memory включаем долгосрочную память.

С включённой памятью агент будет помнить имя пользователя, его предпочтения и историю общения между сессиями. Это превращает бота в настоящего персонального ассистента.
Шаг 5: Onboarding — приветствие и стартовые вопросы
📝 Пример Onboarding
Приветствие:
Привет! Я ваш деловой ассистент. Помогу с перепиской, анализом и планированием. С чего начнём?

Starter questions:
• Напиши коммерческое предложение
• Найди актуальные новости по теме
• Помоги спланировать неделю
Шаг 6: Publish — публикуем агента
  1. Нажмите кнопку Publish
  2. Выберите платформу: Telegram (самый популярный выбор)
  3. Следуйте инструкции по подключению Telegram Bot Token
  4. Поделитесь ссылкой на бота с коллегами или студентами
Подробная инструкция по подключению к Telegram — в разделе ниже.

Workflow — автоматизация без кода

Workflow в Coze позволяет создать цепочку действий. Например: получить запрос → найти в интернете → проанализировать → сформировать отчёт → отправить в Telegram. Всё это через визуальный редактор — без единой строки кода.

Пример Workflow: "Ежедневный дайджест новостей"

Шаг Действие Описание
Trigger Расписание Каждое утро в 8:00 автоматически запускается
Step 1 Web Search Ищет новости по заданным темам за последние 24 часа
Step 2 AI Node Обрабатывает результаты: выделяет главное, убирает лишнее
Step 3 Send to Telegram Отправляет готовый дайджест в Telegram-бот

Публикация в Telegram — пошаговая инструкция

Шаг 1: Создаём бота в Telegram
  1. Откройте Telegram и найдите @BotFather
  2. Отправьте команду /newbot
  3. Введите имя бота (например: "Мой Ассистент")
  4. Введите username бота (например: my_assistant_bot)
  5. Скопируйте Bot Token — он выглядит как: 1234567890:ABCdef...
Шаг 2: Подключаем к Coze
  1. В Coze нажмите Publish
  2. Выберите Telegram
  3. Вставьте скопированный Bot Token
  4. Нажмите Connect
  5. Откройте Telegram и найдите своего бота — он уже работает!
Готово! Теперь у вас есть персональный AI-бот в Telegram. Поделитесь ссылкой с коллегами или клиентами — они смогут пользоваться им бесплатно.
