Groq — Сверхбыстрая инференция

Groq использует специализированные процессоры (LPU) для генерации до 800+ токенов/секунду — в 10-30 раз быстрее обычных GPU. В 2025-2026 добавил поддержку Llama 4, DeepSeek-R2 и Qwen2.5.
Преимущества
  • Очень быстрая генерация
  • Бесплатный tier
  • Поддержка function calling
  • Простая интеграция через API
Ограничения
  • Ограниченный выбор моделей
  • Нет GUI для агентов
  • Лимиты на бесплатном плане
  • Требует программирования

Пример агента на Groq

Python 
from groq import Groq
import json

client = Groq(api_key="your-api-key")

# Определение инструментов
tools = [
    {
        "type": "function",
        "function": {
            "name": "get_weather",
            "description": "Получить текущую погоду в городе",
            "parameters": {
                "type": "object",
                "properties": {
                    "city": {"type": "string", "description": "Название города"}
                },
                "required": ["city"]
            }
        }
    }
]

def run_agent(user_message: str):
    messages = [
        {"role": "system", "content": "Ты - полезный AI агент с доступом к инструментам."},
        {"role": "user", "content": user_message}
    ]

    response = client.chat.completions.create(
        model="llama-4-maverick",  # или deepseek-r2, qwen-2.5-72b
        messages=messages,
        tools=tools,
        tool_choice="auto"
    )

    return response.choices[0].message.content

# Использование
result = run_agent("Какая погода в Минске?")
print(result)

Google Gemini 2.5 — Мультимодальные агенты

Gemini 2.5 — семейство мультимодальных моделей от Google с нативным reasoning. Работает с текстом, изображениями, аудио, видео и кодом. Контекст до 1M токенов. Gemini 2.5 Pro — лучший в кодинге и аналитике по бенчмаркам 2026 года.

Модели Gemini

Модель Особенности Контекст
Gemini 2.5 Flash Быстрая и дешёвая для высоких нагрузок 1M токенов
Gemini 2.5 Pro Лучший в кодинге и аналитике, нативный reasoning 1M токенов
Gemini 2.5 Deep Research Автономный исследователь — сам ищет и анализирует 1M токенов

Пример мультимодального агента

Python 
import google.generativeai as genai
import PIL.Image

genai.configure(api_key="your-api-key")

def multimodal_agent(text: str, image_path: str = None):
    if image_path:
        img = PIL.Image.open(image_path)
        model = genai.GenerativeModel("gemini-2.5-pro")
        response = model.generate_content([text, img])
    else:
        model = genai.GenerativeModel("gemini-2.5-flash")
        response = model.generate_content(text)

    return response.text

# Анализ изображения
result = multimodal_agent(
    "Что изображено на этом фото? Опиши детально.",
    "photo.jpg"
)
print(result)

Другие платформы

OpenRouter — Агрегатор моделей

Единый API для доступа к множеству моделей:

  • GPT-5.2, Claude Sonnet 5, Llama 4 и другие
  • Бесплатный tier для некоторых моделей
  • Поддержка function calling
  • Унифицированный формат запросов

Ссылка: openrouter.ai

Together AI — Open-source модели

Платформа для запуска open-source моделей:

  • Бесплатный tier
  • Быстрая инференция
  • Поддержка function calling
  • Llama, Mistral, и другие модели

Ссылка: together.ai

LangChain / LangGraph — Фреймворки

Open-source фреймворки для создания агентов:

  • Полный контроль над агентом
  • Интеграция с различными моделями
  • Гибкая настройка
  • Требует программирования (Python)

Ссылка: langchain.com

Flowise — No-code LLM приложения

Визуальный редактор для LLM приложений:

  • Drag-and-drop интерфейс
  • Бесплатная self-hosted версия
  • Интеграция с LangChain
  • Поддержка агентов и RAG

Ссылка: flowise.ai

Make.com — Визуальная автоматизация

Make.com (бывший Integromat) — мощная альтернатива n8n с интуитивным визуальным редактором. 1000+ интеграций, AI модули для Claude и GPT, бесплатный план 1000 операций/месяц.
