Режим студента

Агентные системы и автоматизация

Создавайте интеллектуальных AI помощников и автоматизируйте бизнес-процессы.
Custom GPTs, Claude Projects, n8n workflows и альтернативные платформы.

Заметка для преподавателя

Общая продолжительность модуля: ~3 часа (5 уроков + домашнее задание)

Рекомендация: Демонстрируйте создание агентов в реальном времени. Студенты должны создать минимум 2 агента и 1 workflow.

Ключевая цель: Студенты понимают концепцию AI агентов и умеют создавать их на различных платформах для автоматизации бизнес-задач.

Что вы изучите в этом модуле:
  • Концепция AI агентов — типы, компоненты, архитектура
  • Custom GPTs — создание специализированных агентов в ChatGPT
  • Claude Projects — работа с долгосрочной памятью и контекстом
  • n8n — автоматизация workflows с AI интеграцией
  • Альтернативные платформы — Groq, Gemini, Coze и другие

🤖 Главная мысль модуля

"AI агенты — это не просто чат-боты, а автономные системы,
способные использовать инструменты, принимать решения и выполнять задачи."

Финальное задание

Обзор платформ

🎯

ChatGPT Custom GPTs

Создание специализированных агентов с собственными инструкциями, знаниями и интеграциями. Идеально для быстрого прототипирования.

chatgpt.com
📚

Claude Projects

Долгосрочные проекты с расширенным контекстом до 200K токенов. Отлично подходит для сложных аналитических задач.

claude.ai
⚡

n8n Workflows

Open-source платформа для автоматизации с 200+ интеграциями. Визуальный редактор без программирования.

n8n.io
🚀

Groq

Сверхбыстрая инференция LLM (до 300 токенов/сек). Бесплатный tier для экспериментов с open-source моделями.

groq.com
🌈

Google Gemini

Мультимодальные агенты с поддержкой текста, изображений, аудио и видео. Function calling из коробки.

gemini.google.com
Быстрый старт

Для успешного прохождения модуля вам понадобится:

  • Аккаунт ChatGPT (Plus для Custom GPTs)
  • Аккаунт Claude.ai (Pro для Projects)
  • Установленный n8n (локально или облачно)
  • API ключи OpenAI или Anthropic
  • Аккаунт Groq (бесплатный)
