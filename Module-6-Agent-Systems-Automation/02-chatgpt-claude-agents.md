# Создание агентов в ChatGPT и Claude

## Введение

В этом разделе мы рассмотрим практические аспекты создания AI агентов на двух популярных платформах: ChatGPT (Custom GPTs) и Claude (Projects). Каждая платформа имеет свои особенности и преимущества, которые мы изучим на практических примерах.

## ChatGPT Custom GPTs

### Что такое Custom GPTs?

Custom GPTs — это персонализированные версии ChatGPT, которые можно настроить для выполнения специфических задач. Они позволяют создавать специализированных агентов с собственными инструкциями, знаниями и возможностями.

### Преимущества Custom GPTs

- **Простота создания**: интуитивный интерфейс без программирования
- **Встроенные инструменты**: поиск в интернете, генерация изображений, анализ кода
- **Управление знаниями**: загрузка файлов и документов
- **Публикация**: возможность делиться GPT с другими пользователями
- **Интеграция**: работа через API и веб-интерфейс

### Пошаговое создание Custom GPT

#### Шаг 1: Создание нового GPT

1. Откройте ChatGPT
2. В левом меню нажмите "Explore"
3. Выберите "Create a GPT"
4. Нажмите "Create" в разделе "Create a new GPT"

#### Шаг 2: Настройка базовой информации

**Название GPT**:
```
Маркетинговый ассистент
```

**Описание**:
```
Эксперт по маркетингу, который помогает создавать контент для социальных сетей, 
планировать кампании и анализировать эффективность. Специализируется на 
B2B и B2C маркетинге, контент-стратегии и digital-маркетинге.
```

**Инструкции (Instructions)**:
```
Ты - опытный маркетолог с 10+ летним стажем работы в digital-маркетинге. 
Твоя задача - помогать создавать эффективный контент и стратегии для различных платформ.

Твои основные функции:
1. Создание контента для социальных сетей (LinkedIn, Facebook, Instagram, Twitter)
2. Разработка email-кампаний и рассылок
3. Планирование контент-стратегий
4. Анализ трендов и конкурентов
5. Создание рекламных текстов и объявлений

Принципы работы:
- Всегда учитывай целевую аудиторию
- Адаптируй тон и стиль под платформу
- Предлагай конкретные и actionable решения
- Используй данные и аналитику для обоснования рекомендаций
- Следи за актуальными трендами в маркетинге

Формат ответов:
- Структурированные и легко читаемые
- С конкретными примерами
- С практическими советами
- С метриками для измерения эффективности
```

#### Шаг 3: Настройка поведения

**Conversation starters**:
```
1. "Создай контент-план на неделю для LinkedIn"
2. "Напиши email-рассылку для привлечения новых клиентов"
3. "Проанализируй тренды в маркетинге на 2024 год"
4. "Создай рекламное объявление для Facebook"
5. "Помоги оптимизировать контент для SEO"
```

**Capabilities**:
- ✅ **Web Search**: Включено (для поиска актуальной информации)
- ✅ **Code Interpreter**: Включено (для анализа данных)
- ✅ **DALL-E**: Включено (для создания изображений)

**Knowledge**: Загрузите файлы с примерами:
- Примеры успешных кампаний
- Шаблоны контента
- Гайдлайны брендов
- Аналитические отчеты

#### Шаг 4: Тестирование GPT

**Тестовый запрос**:
```
Создай пост для LinkedIn о важности автоматизации в маркетинге. 
Целевая аудитория: маркетологи и владельцы малого бизнеса. 
Цель: привлечь внимание к услугам автоматизации маркетинга.
```

**Ожидаемый результат**:
- Структурированный пост с заголовком
- Основной текст с конкретными примерами
- Call-to-action
- Рекомендации по хештегам
- Предложения по визуальному оформлению

### Продвинутые настройки Custom GPT

#### Использование Actions (API интеграции)

Custom GPTs поддерживают интеграцию с внешними API через Actions. Это позволяет агентам взаимодействовать с внешними сервисами.

**Пример настройки Action**:
```json
{
  "openapi": "3.0.0",
  "info": {
    "title": "Weather API",
    "version": "1.0.0"
  },
  "servers": [
    {
      "url": "https://api.weather.com"
    }
  ],
  "paths": {
    "/weather": {
      "get": {
        "summary": "Get weather information",
        "parameters": [
          {
            "name": "city",
            "in": "query",
            "required": true,
            "schema": {
              "type": "string"
            }
          }
        ],
        "responses": {
          "200": {
            "description": "Weather information",
            "content": {
              "application/json": {
                "schema": {
                  "type": "object",
                  "properties": {
                    "temperature": {"type": "number"},
                    "condition": {"type": "string"}
                  }
                }
              }
            }
          }
        }
      }
    }
  }
}
```

#### Настройка контекста и памяти

Custom GPTs могут сохранять контекст разговора, но не имеют долгосрочной памяти между сессиями. Для работы с контекстом:

1. **Используйте четкие инструкции** о том, что нужно запомнить
2. **Загружайте файлы с контекстом** в Knowledge
3. **Структурируйте разговор** с явными ссылками на предыдущую информацию

## Claude Projects

### Что такое Claude Projects?

Claude Projects — это функция Claude, которая позволяет создавать долгосрочные проекты с расширенным контекстом, управлением файлами и памятью. Это идеально подходит для сложных задач, требующих работы с большими объемами данных.

### Преимущества Claude Projects

- **Долгосрочная память**: сохранение контекста между сессиями
- **Работа с файлами**: загрузка и анализ документов
- **Расширенный контекст**: до 200K токенов
- **Управление проектами**: организация работы по темам
- **Коллаборация**: совместная работа над проектами

### Создание Claude Project

#### Шаг 1: Создание нового проекта

1. Откройте Claude.ai
2. Нажмите "New Project"
3. Введите название проекта
4. Добавьте описание проекта

#### Шаг 2: Настройка проекта

**Название проекта**:
```
Аналитический ассистент для e-commerce
```

**Описание**:
```
Проект для анализа данных интернет-магазина, создания отчетов 
и рекомендаций по оптимизации продаж. Включает анализ трафика, 
конверсии, поведения клиентов и эффективности маркетинговых кампаний.
```

**System Instructions**:
```
Ты - эксперт по аналитике e-commerce с опытом работы в различных 
интернет-магазинах. Твоя задача - анализировать данные и создавать 
actionable insights для улучшения бизнес-показателей.

Твои основные функции:
1. Анализ данных о продажах и трафике
2. Создание отчетов и дашбордов
3. Выявление трендов и паттернов
4. Рекомендации по оптимизации
5. Прогнозирование и планирование

Принципы работы:
- Используй конкретные данные и метрики
- Предлагай практические рекомендации
- Учитывай специфику e-commerce
- Фокусируйся на ROI и конверсии
- Создавай понятные визуализации

Формат отчетов:
- Executive Summary
- Ключевые метрики
- Анализ трендов
- Рекомендации
- План действий
```

#### Шаг 3: Загрузка данных

Claude Projects позволяет загружать различные типы файлов:

**Поддерживаемые форматы**:
- **Текстовые**: .txt, .md, .csv
- **Документы**: .pdf, .docx, .pptx
- **Данные**: .xlsx, .json, .xml
- **Изображения**: .png, .jpg, .gif
- **Код**: .py, .js, .html, .css

**Пример загрузки данных**:
```
Загрузите файлы:
1. sales_data_2024.xlsx - данные о продажах
2. traffic_analytics.csv - аналитика трафика
3. customer_feedback.pdf - отзывы клиентов
4. marketing_campaigns.json - данные о кампаниях
```

#### Шаг 4: Настройка памяти

Claude Projects автоматически сохраняет контекст, но можно настроить дополнительную память:

**Типы памяти**:
- **Контекстная**: информация о текущем проекте
- **Фактическая**: данные и метрики
- **Процедурная**: алгоритмы и методы анализа
- **Эпизодическая**: результаты предыдущих анализов

### Интеграция с внешними сервисами

Claude Projects можно интегрировать с внешними сервисами через API:

#### Пример интеграции с Google Sheets

```python
# Пример кода для интеграции с Google Sheets
import gspread
from google.oauth2.service_account import Credentials

def update_analytics_data(project_data):
    """Обновление данных аналитики в Google Sheets"""
    
    # Настройка подключения
    scope = ['https://spreadsheets.google.com/feeds',
             'https://www.googleapis.com/auth/drive']
    creds = Credentials.from_service_account_file('credentials.json', scopes=scope)
    client = gspread.authorize(creds)
    
    # Открытие таблицы
    sheet = client.open('Analytics Dashboard').sheet1
    
    # Обновление данных
    for row in project_data:
        sheet.append_row(row)
    
    return "Данные успешно обновлены"
```

#### Пример интеграции с Slack

```python
# Пример уведомления в Slack
import requests

def send_slack_notification(message, channel="#analytics"):
    """Отправка уведомления в Slack"""
    
    webhook_url = "YOUR_SLACK_WEBHOOK_URL"
    
    payload = {
        "text": message,
        "channel": channel
    }
    
    response = requests.post(webhook_url, json=payload)
    return response.status_code == 200
```

## Сравнение платформ

### ChatGPT Custom GPTs vs Claude Projects

| Критерий | Custom GPTs | Claude Projects |
|----------|-------------|-----------------|
| **Простота создания** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| **Гибкость настройки** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **Работа с файлами** | ⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| **Долгосрочная память** | ⭐⭐ | ⭐⭐⭐⭐⭐ |
| **API интеграции** | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| **Публикация** | ⭐⭐⭐⭐⭐ | ⭐⭐ |
| **Стоимость** | Бесплатно/Платно | Бесплатно/Платно |

### Когда использовать Custom GPTs

- Простые задачи с четкими инструкциями
- Публичные агенты для широкой аудитории
- Быстрое прототипирование
- Задачи, не требующие работы с большими файлами

### Когда использовать Claude Projects

- Сложные аналитические задачи
- Работа с большими объемами данных
- Долгосрочные проекты
- Задачи, требующие глубокого контекста

## Альтернативные платформы

Помимо ChatGPT Custom GPTs и Claude Projects, существует множество других платформ для создания AI агентов:

- **Groq**: Быстрая инференция LLM с бесплатным tier (см. `04-alternative-agents-platforms.md`)
- **Google Gemini**: Мультимодальные агенты с function calling (см. `04-alternative-agents-platforms.md`)
- **Coze**: Бесплатная no-code платформа для создания агентов (см. `04-alternative-agents-platforms.md`)
- **OpenRouter**: Агрегатор моделей через единый API
- **Together AI**: Платформа для open-source моделей
- **LangChain/LangGraph**: Open-source фреймворки для агентов
- **Flowise**: No-code платформа с визуальным редактором

Подробнее об этих платформах и их использовании см. в файле `04-alternative-agents-platforms.md`.
