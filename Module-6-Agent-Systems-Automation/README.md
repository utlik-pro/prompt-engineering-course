# Модуль 6: Агентные системы и автоматизация

## Описание модуля

Этот модуль посвящен изучению AI агентов и их практическому применению для автоматизации бизнес-процессов. Участники научатся создавать интеллектуальных помощников, которые могут работать автономно или под управлением человека.

## Структура модуля

### Файлы модуля

- `00-module-overview.md` - Обзор модуля и цели обучения
- `01-ai-agents-concepts.md` - Концепция AI агентов и их типы
- `02-chatgpt-claude-agents.md` - Создание агентов в ChatGPT и Claude
- `03-n8n-automation.md` - Введение в n8n и автоматизацию
- `04-alternative-agents-platforms.md` - Альтернативные платформы (Groq, Gemini, Coze, и др.)
- `training-program-3h.md` - 3-часовая программа обучения

## Цели обучения

После прохождения модуля участники смогут:

1. **Понимать концепцию AI агентов**
   - Различать типы агентов и их применение
   - Понимать компоненты и архитектуру агентов
   - Оценивать возможности и ограничения

2. **Создавать агентов в популярных платформах**
   - Настраивать Custom GPTs в ChatGPT
   - Работать с Claude Projects для сложных задач
   - Использовать встроенные инструменты и интеграции

3. **Автоматизировать процессы с помощью n8n**
   - Создавать workflows для автоматизации
   - Интегрировать AI через API
   - Настраивать триггеры и действия

4. **Применять агентов в реальных задачах**
   - Решать конкретные бизнес-задачи
   - Интегрировать различные сервисы
   - Измерять эффективность решений

## Инструменты и платформы

### Основные платформы:
- **ChatGPT**: Custom GPTs для создания агентов
- **Claude.ai**: Projects для работы с контекстом
- **n8n**: автоматизация и интеграции
- **Make.com**: альтернативная платформа автоматизации

### Альтернативные платформы (см. `04-alternative-agents-platforms.md`):
- **Groq**: быстрая инференция LLM с бесплатным tier
- **Google Gemini**: мультимодальные агенты с function calling
- **Coze**: бесплатная no-code платформа для создания агентов
- **OpenRouter**: агрегатор моделей через единый API
- **Together AI**: платформа для open-source моделей
- **LangChain/LangGraph**: open-source фреймворки для агентов
- **Flowise**: no-code платформа с визуальным редактором

### Дополнительные инструменты:
- **OpenAI API**: для кастомных интеграций
- **Telegram Bot API**: для тестирования
- **Google Sheets API**: для работы с данными
- **Webhook**: для интеграций
