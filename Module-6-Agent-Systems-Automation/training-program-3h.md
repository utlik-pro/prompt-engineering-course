### 1.1 Теория: Концепция AI агентов (20 минут)

**Ключевые определения:**

```
AI агент = автономная система, способная воспринимать окружение,
принимать решения и выполнять действия для достижения целей
```

**Типы AI агентов:**

1. **Автономные агенты**
   - Работают независимо без постоянного контроля
   - Принимают решения на основе встроенных правил
   - Примеры: торговые боты, автономные автомобили

2. **Управляемые агенты**
   - Работают под контролем человека
   - Выполняют конкретные задачи по запросу
   - Примеры: ChatGPT Custom GPTs, Claude Projects

3. **Гибридные агенты**
   - Сочетают автономность и управляемость
   - Могут переключаться между режимами
   - Примеры: n8n workflows с AI нодами

**Компоненты AI агента:**

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   Восприятие    │───▶│   Обработка     │───▶│    Действие     │
│   (Input)       │    │   (Processing)  │    │   (Output)      │
└─────────────────┘    └─────────────────┘    └─────────────────┘
         │                       │                       │
         ▼                       ▼                       ▼
   - Текстовые запросы      - LLM обработка        - Ответы
   - Изображения           - Анализ контекста      - Действия
   - Данные                - Принятие решений      - Интеграции
```

### 1.2 Tool use и function calling (15 минут)

**Что такое Tool use:**

```
Tool use = способность AI агента использовать внешние инструменты
для выполнения задач, которые он не может решить самостоятельно
```

**Типы инструментов:**

1. **API интеграции**
   - Поиск в интернете
   - Работа с базами данных
   - Отправка уведомлений

2. **Файловые операции**
   - Чтение и создание файлов
   - Обработка изображений
   - Работа с документами

3. **Вычисления**
   - Математические расчеты
   - Анализ данных
   - Генерация кода

def send_email(to: str, subject: str, body: str) -> bool:
    """Отправка email"""

### 4.1 Знакомство с n8n (15 минут)

**Что такое n8n:**

```
n8n = open-source платформа для автоматизации workflows
Позволяет соединять различные сервисы и API без программирования
```

**Основные концепции:**

1. **Workflow** - последовательность действий
2. **Node** - отдельный шаг в workflow
3. **Connection** - связь между нодами
4. **Trigger** - событие, запускающее workflow
5. **Action** - действие, выполняемое в workflow

**Типы нодов:**

- **Triggers**: Webhook, Schedule, Email, Manual
- **AI/ML**: OpenAI, HuggingFace, HTTP Request
- **Data**: Set, Code, Function
- **Communication**: Telegram, Discord, Email, Slack
- **Storage**: Google Sheets, Airtable, Database

**Интерфейс n8n:**

```
┌─────────────────────────────────────────────────────────┐
│  [Menu]  [Workflows]  [Executions]  [Settings]         │
├─────────────────────────────────────────────────────────┤
│  ┌─────────┐    ┌─────────┐    ┌─────────┐             │
│  │ Trigger │───▶│   AI    │───▶│ Action  │             │
│  │  Node   │    │  Node   │    │  Node   │             │
│  └─────────┘    └─────────┘    └─────────┘             │
│                                                         │
│  [Execute Workflow]  [Save]  [Settings]                │
└─────────────────────────────────────────────────────────┘
```

### 4.2 Создание первого workflow (25 минут)

**Задача: Создать AI-ассистента для Telegram**

**Шаг 1: Настройка Telegram бота (5 минут)**

1. Создайте бота через @BotFather в Telegram
2. Получите токен бота
3. Запомните username бота

**Шаг 2: Создание workflow в n8n (10 минут)**

```
Workflow: "Telegram AI Assistant"

1. Trigger Node: Telegram Trigger
   - Event: "Message"
   - Bot Token: [ваш токен]

2. AI Node: OpenAI
   - Model: "gpt-3.5-turbo"
   - Messages: 
     - System: "Ты - полезный ассистент. Отвечай кратко и по делу."
     - User: "{{ $json.message.text }}"

3. Action Node: Telegram Send Message
   - Chat ID: "{{ $json.message.chat.id }}"
   - Text: "{{ $json.choices[0].message.content }}"
```

**Шаг 3: Настройка OpenAI API (5 минут)**

1. Получите API ключ на platform.openai.com
2. Добавьте ключ в n8n Credentials
3. Настройте нод OpenAI

**Шаг 4: Тестирование (5 минут)**

1. Активируйте workflow
2. Отправьте сообщение боту в Telegram
3. Проверьте ответ

**Расширенный workflow с памятью:**

```
1. Telegram Trigger
2. Set Node (сохранить сообщение)
3. Code Node (добавить в контекст)
4. OpenAI (с контекстом)
5. Code Node (обновить контекст)
6. Telegram Send Message
```
