## Модуль 3: Продвинутые техники

### Уроки

1. **Chain-of-Thought (CoT)** — пошаговое рассуждение
2. **Tree of Thoughts (ToT)** — дерево вариантов
3. **Self-Consistency** — множественные запуски
4. **ReAct** — рассуждение + действие
5. **Практика CoT** — 4 задачи на логику
6. **Практика продвинутые** — ToT, Self-Consistency, ReAct
7. **Домашнее задание** — 5 обязательных задач

### Функции

- Интерактивные демо для тестирования промптов
- Квизы с обратной связью
- Чек-листы для практики
- Прогресс-трекинг
- Режим студента/преподавателя

## Для преподавателя

### Вход в админ-режим

- **5 кликов** по логотипу "PromptMaster"
- Или **Ctrl + Shift + A**
- **Пароль:** `609285`

### Возможности

- Видеть заметки для преподавателя с таймингом
- Разблокировать все уроки
- Сбросить прогресс студентов
- Переключаться между режимами

## Технические детали

- **Без сервера** — работает локально в браузере
- **LocalStorage** — сохранение прогресса
- **Адаптивный дизайн** — работает на мобильных
