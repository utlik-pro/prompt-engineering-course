В этом уроке мы переходим от теории к практике. Вы напишете свои первые промпты для 5 типичных сценариев использования AI. Откройте любую AI-платформу (ChatGPT, Claude или Gemini) и выполняйте задания параллельно.

Заметки для преподавателя

Формат урока: Практический воркшоп. Студенты работают с реальными AI-платформами. После каждого задания обсудите результаты — у разных студентов будут разные ответы.

Время: ~8 минут на задание + обсуждение.

Задание 1: Генерация текста

Сценарий: Email для клиента

Ситуация: Вы менеджер по работе с клиентами. Клиент недоволен задержкой доставки заказа на 3 дня. Нужно написать извинительное письмо.

Ваша задача: Напишите промпт для создания извинительного email-письма.
Показать пример промпта
Пример промпта 
Контекст: Я менеджер по работе с клиентами интернет-магазина электроники.
Клиент заказал ноутбук, но доставка задержалась на 3 дня из-за
проблем с логистикой.

Роль: Выступи в роли опытного специалиста по клиентскому сервису.

Задача: Напиши извинительное email-письмо клиенту.

Формат:
- Тема письма
- Приветствие
- Извинение и объяснение причины
- Компенсация (предложи скидку 10% на следующий заказ)
- Заключение

Тон: Вежливый, эмпатичный, профессиональный.
Длина: 150-200 слов.
Критерии успеха:
  • Письмо содержит искреннее извинение
  • Указана причина задержки
  • Предложена компенсация
  • Тон профессиональный и эмпатичный

Задание 2: Суммаризация

Сценарий: Краткое изложение статьи

Ситуация: Вам нужно подготовить краткий обзор длинной статьи для руководства. Времени читать полностью нет.

Ваша задача: Напишите промпт для создания структурированного резюме любой статьи.
Показать пример промпта
Пример промпта 
Создай структурированное резюме следующей статьи.

Формат вывода:
1. Заголовок (1 строка)
2. Главная идея (2-3 предложения)
3. Ключевые тезисы (3-5 пунктов)
4. Важные цифры/факты (если есть)
5. Вывод автора (1-2 предложения)

Ограничения:
- Общая длина резюме: не более 200 слов
- Используй маркированные списки
- Сохрани объективность, не добавляй свои оценки

Статья:
[вставьте текст статьи]
Тестовый текст для практики:

Найдите любую статью о технологиях на Habr, Medium или в новостях и попробуйте ваш промпт на ней.

Задание 3: Креативный контент

Сценарий: Пост для социальных сетей

Ситуация: Вы SMM-менеджер кофейни. Нужно написать пост о новом сезонном напитке — тыквенном латте.

Ваша задача: Напишите промпт для создания вовлекающего поста.
Показать пример промпта
Пример промпта 
Роль: Ты креативный SMM-менеджер уютной кофейни в центре города.

Задача: Напиши пост для Instagram о запуске нового сезонного напитка —
тыквенного латте с корицей.

Требования:
- Длина: 100-150 слов
- Стиль: тёплый, уютный, вызывающий желание попробовать
- Включи эмодзи (3-5 штук, уместно)
- Добавь призыв к действию
- Предложи 3 хэштега

Дополнительно:
- Напиток доступен только в октябре-ноябре
- Цена: 350 рублей
- Можно добавить взбитые сливки бесплатно

Целевая аудитория: молодые профессионалы 22-35 лет,
любящие уют и сезонные напитки.

Задание 4: Анализ и рекомендации

Сценарий: Анализ конкурентов

Ситуация: Вы планируете открыть онлайн-школу английского языка. Нужно проанализировать конкурентное преимущество.

Ваша задача: Напишите промпт для получения структурированного анализа.
Показать пример промпта
Пример промпта 
Роль: Ты бизнес-консультант с опытом в EdTech.

Контекст: Я планирую запустить онлайн-школу английского языка
для взрослых (25-45 лет), с фокусом на бизнес-английский.

Задача: Проведи анализ и дай рекомендации.

Структура ответа:
1. Основные конкуренты на рынке (3-5 игроков)
2. Типичные ценовые модели в нише
3. Какие УТП (уникальные торговые предложения) работают
4. Потенциальные ниши, которые не заняты
5. 3 конкретные рекомендации для входа на рынок

Формат: Структурированный отчёт с заголовками и буллитами.
Длина: 400-500 слов.

Важно: Основывайся на реальных трендах рынка онлайн-образования.
Важно: AI может не знать актуальных данных о ценах конкурентов. Используйте ответ как отправную точку для собственного исследования.

Задание 5: Решение проблемы

Сценарий: Brainstorm идей

Ситуация: Команда работает удалённо, и вовлечённость сотрудников упала. Нужны идеи для улучшения командной работы.

Ваша задача: Напишите промпт для генерации креативных решений.
Показать пример промпта
Пример промпта 
Роль: Ты HR-консультант с опытом в удалённых командах.

Контекст:
- IT-компания, 30 сотрудников
- Полностью удалённая работа уже 2 года
- Проблема: снижение вовлечённости, люди чувствуют себя изолированными
- Бюджет на team-building: ограничен

Задача: Предложи 10 идей для повышения вовлечённости удалённой команды.

Требования к идеям:
- 5 бесплатных/малобюджетных
- 3 среднего бюджета
- 2 премиум-идеи

Для каждой идеи укажи:
- Название
- Краткое описание (1-2 предложения)
- Примерные затраты
- Ожидаемый эффект

Формат: Нумерованный список с подпунктами.

Как улучшать промпты

Техника итеративного улучшения
1
Запустите первую версию

Не пытайтесь написать идеальный промпт сразу. Начните с базовой версии.

2
Оцените результат

Что хорошо? Что не так? Чего не хватает?

3
Уточните в диалоге

"Сделай вступление короче", "Добавь больше конкретики", "Измени тон на более формальный"

4
Сохраните финальную версию

Когда результат устраивает — сохраните промпт для повторного использования.
