Задание 1: Создание защищённого промпта (30 мин)

🛡️ Защищённый чатбот поддержки

Создайте максимально защищённый промпт для чатбота службы поддержки интернет-магазина.

Функциональные требования:

  • Отвечает на вопросы о товарах и услугах
  • Помогает с оформлением заказа
  • Обрабатывает запросы на возврат
  • Предоставляет контактную информацию
  • Направляет к оператору при необходимости

Требования безопасности:

  • Защита от всех типов injection (direct, indirect, multi-turn)
  • Защита от jailbreaking
  • Защита системного промпта от раскрытия
  • Защита конфиденциальных данных
  • Вежливый отказ при подозрительных запросах

Что включить в ответ:

  1. Полный текст системного промпта
  2. Описание применённых техник защиты
  3. Тест-кейсы для проверки защиты (минимум 5)
  4. Скриншоты успешного отражения атак

Задание 2: Red Teaming (20 мин)

🔴 Этичное тестирование на проникновение

Протестируйте промпт одногруппника (с его/её разрешения) и составьте отчёт о найденных уязвимостях.

Важно: Тестируйте ТОЛЬКО с явного разрешения владельца промпта!

Что сделать:

  1. Получите промпт одногруппника для тестирования
  2. Попробуйте минимум 10 различных атак
  3. Задокументируйте каждую попытку
  4. Составьте отчёт с рекомендациями

Типы атак для тестирования:

  • Direct Injection (минимум 3 варианта)
  • Role Manipulation (минимум 2 варианта)
  • Information Extraction (минимум 2 варианта)
  • Multi-turn Attack (минимум 1 сценарий)
  • Jailbreaking (минимум 2 техники)
