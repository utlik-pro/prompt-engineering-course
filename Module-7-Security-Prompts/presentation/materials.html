🧙‍♂️ Gandalf — учебная игра

Lakera Gandalf

Что это: бесплатная браузерная игра, в которой LLM-бот хранит секретный пароль на каждом уровне. Задача игрока — выдавить пароль с помощью промпт-инъекций. С каждым уровнем защита усиливается.

  • 🔗 Игра: gandalf.lakera.ai
  • 🔗 Бонус-уровень: gandalf.lakera.ai/adventure-8
  • 🔗 Компания-создатель: lakera.ai

Уровней: 7 основных + 1 бонус. Регистрации не требуется. Подробный разбор каждого уровня — в Уроке 4.

jailbreak_llms (verazuo)

🛡️ Стандарты и фреймворки

OWASP Top 10 для LLM-приложений

Отраслевой стандарт описания топ-10 рисков LLM-систем. Версия 2025 — основной справочник для production-проектов.

🔗 owasp.org/www-project-top-10-for-large-language-model-applications

LLM01 Prompt Injection — то, чему посвящён наш модуль — в этом списке номер один с момента публикации.

MITRE ATLAS

Taxonomy атак на ML/LLM-системы — аналог MITRE ATT&CK, но для AI. Для тех, кто работает с энтерпрайз-безопасностью.

🔗 atlas.mitre.org

NIST AI Risk Management Framework

Государственный стандарт США по управлению рисками AI. Generative AI Profile (AI 600-1) — раздел, релевантный для LLM.

🔗 nist.gov/itl/ai-risk-management-framework

🔧 Инструменты тестирования (для углублённого изучения)

Готовые инструменты для автоматического red-team'инга промптов. В рамках курса не обязательны — но если хотите углубиться после модуля, начните отсюда.

  • Llama Guard (Meta) — open-source классификатор контента. huggingface.co/meta-llama/LlamaGuard-7b
  • Lakera Guard — коммерческий SaaS-фильтр от создателей Gandalf. lakera.ai/lakera-guard

📑 Ключевые статьи (для самых любопытных)

Если уровень модуля стал тесным — копайте эти работы. Все — open access.

  • «Not what you've signed up for: Compromising Real-World LLM-Integrated Applications with Language Prompts» (Greshake et al., 2023) — статья, ввёдшая термин indirect prompt injection. arxiv.org/abs/2302.12173
  • «Universal and Transferable Adversarial Attacks on Aligned Language Models» (Zou et al., 2023) — про GCG-атаки (универсальные суффиксы). arxiv.org/abs/2307.15043
  • «Jailbreaking Black Box LLMs in Twenty Queries» (Chao et al., 2023) — PAIR-атаки, автоматизированный jailbreak. arxiv.org/abs/2310.08419
  • «Many-shot Jailbreaking» (Anthropic, 2024) — атака через длинный контекст. anthropic.com/research/many-shot-jailbreaking
