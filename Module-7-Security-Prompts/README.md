# Модуль 7: Безопасность и защита промптов

## Обзор модуля

Этот модуль посвящен изучению безопасности промптов и защите от различных типов атак, включая prompt injection и jailbreaking. Участники научатся выявлять уязвимости, создавать защищенные промпты и применять best practices для обеспечения безопасности AI-систем.

## Ключевые темы

### Теория (1 час)
- Prompt injection и jailbreaking
- Типы атак: direct, indirect, multi-turn
- Техники защиты промптов
- Этические вопросы и границы
- Data privacy и конфиденциальность

### Практика (1.5 часа)
- Тестирование уязвимостей промптов
- Создание защищенных промптов
- Red teaming: этичные попытки обойти ограничения
- Анализ реальных кейсов prompt injection

### Домашнее задание (0.5 часа)
- Создание максимально защищенного промпта для чатбота поддержки
- Этичные попытки "взломать" промпты одногруппников

## Инструменты

- ChatGPT (для тестирования)
- Claude (для тестирования)
- Gandalf game (lakera.ai)
- OWASP LLM Top 10 (руководство)

## Целевая аудитория

- Разработчики AI-приложений
- Prompt engineers
- Security specialists
- Product managers, работающие с AI
- Все, кто создает AI-системы для production

## Предварительные требования

- Базовое понимание промпт-инжиниринга
- Опыт работы с ChatGPT или Claude
- Понимание основ безопасности (желательно)

## Ожидаемые результаты

После прохождения модуля участники смогут:

1. Понимать различные типы атак на промпты
2. Выявлять уязвимости в промптах
3. Создавать защищенные промпты
4. Применять best practices безопасности
5. Проводить этичное тестирование на проникновение
