Chain-of-Thought: Думай шаг за шагом

Цели урока:

  • Понять, почему AI ошибается в сложных задачах
  • Изучить технику пошагового рассуждения (CoT)
  • Научиться применять CoT для разных задач
  • Увидеть реальную разницу в качестве ответов
Заметка для преподавателя

Время: 20 минут

Подход: Это самая "научная" техника курса. Покажите математический пример в реальном времени — студенты увидят, как AI "думает" вслух.

Важно: Подчеркните, что CoT не нужен для простых задач. Это инструмент для сложных случаев.

Проблема: AI ошибается в сложных задачах

Языковые модели обучены предсказывать следующее слово. Они не "думают" — они генерируют вероятный текст.

Это приводит к проблеме: в сложных задачах AI может "перепрыгнуть" к ответу, минуя важные шаги рассуждения.

Типичные ошибки: Неправильные расчёты, ошибки в логике, пропущенные факторы, поверхностный анализ.

Решение: Chain-of-Thought (CoT)

Chain-of-Thought

"Давай подумаем шаг за шагом"

Техника, которая заставляет AI показывать ход своих рассуждений

Почему это работает?

Когда AI записывает промежуточные шаги:

  1. Каждый шаг становится частью контекста для следующего
  2. AI "проверяет" себя на каждом этапе
  3. Ошибки легче заметить и исправить
  4. Результат становится более обоснованным
Исследования показывают: CoT повышает точность на 30-40% в задачах на логику и математику.

Пример: математическая задача

Задача

В магазине было 23 яблока. Утром купили 6 яблок, а после обеда привезли ещё 12. Сколько яблок стало в магазине?

Без CoT 
В магазине стало 29 яблок.

❌ Неправильно!
С CoT 
Давай решим шаг за шагом:

1. Изначально было: 23 яблока
2. Утром купили (забрали): 23 - 6 = 17
3. После обеда привезли: 17 + 12 = 29

Ответ: 29 яблок

✓ Правильно!
Заметили? В данном случае ответ одинаковый, но с CoT мы можем проверить логику. В более сложных задачах без CoT ошибки будут неизбежны.

Фразы-триггеры для CoT

Варианты запроса CoT 
Давай подумаем шаг за шагом.

Рассуждай вслух, показывая каждый шаг.

Объясни своё рассуждение перед ответом.

Сначала разбери задачу на части, потом реши.

Проанализируй все факторы по порядку.

Think step by step. (на английском тоже работает)

Когда использовать CoT?

✓ Используйте ✗ Не нужно
Математические расчёты Простые вопросы
Логические задачи Творческое письмо
Многофакторный анализ Перевод текста
Принятие решений Генерация списков

Практические примеры CoT

Пример 1: Бизнес-решение
Промпт с CoT 
У меня стартап: приложение для изучения языков. Бюджет на маркетинг: $5000.
Каналы: Instagram Ads, Google Ads, Influencer Marketing, Content Marketing.

Как распределить бюджет?

Проанализируй каждый канал по критериям: стоимость привлечения,
скорость результата, масштабируемость. Затем дай рекомендацию.
Пример 2: Отладка кода
Промпт с CoT 
Этот код должен сортировать массив, но выдаёт ошибку:

```javascript
function sort(arr) {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] > arr[i+1]) {
      [arr[i], arr[i+1]] = [arr[i+1], arr[i]];
    }
  }
  return arr;
}
```

Пройди по коду строка за строкой. Объясни, что происходит на каждом шаге.
Найди, где возникает ошибка.
Пример 3: Сложный анализ
Промпт с CoT 
Компания: онлайн-магазин одежды.
Проблема: конверсия упала с 3% до 1.5% за месяц.

Что могло случиться?

Рассуждай пошагово:
1. Перечисли все возможные причины падения конверсии
2. Оцени вероятность каждой причины
3. Предложи способ проверки каждой гипотезы
4. Дай приоритизированный план действий

Бонус: Zero-shot CoT

Самый простой способ активировать CoT — добавить в конец промпта:

"Let's think step by step."

Это работает даже без примеров. AI понимает, что нужно показать рассуждение.

Ключевые выводы

  • Chain-of-Thought — техника пошагового рассуждения
  • Ключевая фраза: "Давай подумаем шаг за шагом"
  • Повышает точность на 30-40% в сложных задачах
  • Используйте для: математики, логики, анализа, отладки
  • Не нужно для: простых задач, творчества, переводов
  • Zero-shot CoT: просто добавьте "Let's think step by step"
